# raw_leverage ビン別 × 前方1年 3倍NASDAQリターン分析

## 目的

GAS自動運用システムが日次算出する **raw_leverage**（推奨保有比率の理論値）の各水準において、
「もし3倍NASDAQ投信を1年間保有した場合のリターン」を推計する。
これにより、**各レバレッジ水準での期待リターンを定量化**し、他の投資先との比較材料とする。

---

## 分析条件

| 項目 | 値 |
|---|---|
| データ期間 | 1974-01-02 〜 2021-05-07（47年間） |
| 有効分析期間 | 1975-08-01 〜 2020-05-11（ウォームアップ400日＋前方250日を除外） |
| 有効データ点数 | 11,293日 |
| ベースレバレッジ | 3倍（日次リバランス） |
| 年間コスト | 1.5%（経費率、日割り控除） |
| 前方リターン計算期間 | 250営業日（≒1年） |
| ビニング | raw_leverage を 0〜100% の 10%刻みで分割 |

---

## raw_leverage の算出方法

GAS実装（[NASDAQ-strategy-gas](https://github.com/KazuyaMurayama/NASDAQ-strategy-gas)）と同一パラメータで4層を乗算：

```
raw_leverage = DD × VT × SlopeMult × MomDecel   (clip [0, 1])
```

| レイヤー | 計算内容 | パラメータ | 出力範囲 |
|---|---|---|---|
| **DD** | 200日高値比ヒステリシス | exit=0.82, reentry=0.92 | 0 or 1 |
| **VT** | TrendTV(MA150) / AsymEWMA(↓span5, ↑span20) | TV: 0.15〜0.35 | 0〜1 |
| **SlopeMult** | MA200傾きZスコア | base=0.7, sens=0.3, norm=60日 | 0.3〜1.5 |
| **MomDecel** | 短期(40日)/長期(120日)モメンタム乖離Zスコア | sens=0.3, z_window=120 | 0.5〜1.3 |

## 解釈とインサイト

### 1. 0-10%ビン（CASH指示時）：平均+68.1%だが超ハイリスク

- DD=0（CASH）のときは暴落直後であり、その後の**反発リターンが極めて大きい**
- しかし標準偏差137%、プラス確率66.6%は全ビン中最低 → **3回に1回は損失**
- Q25が-65.8%と深い → 暴落が続く局面で3倍保有は致命的
- **戦略がCASHを指示する合理性を裏付け**：リターンは高いがリスクも最大

### 2. 90-100%ビン（フル投資推奨時）：最も安定

- プラス確率78.1%で**全ビン最高** → 戦略が「GO」と判断した局面の信頼性
- 中央値+40.7%、Q25が+4.8%（プラス圏） → 下位25%でもほぼ損しない
- **他投資との比較基準値**: 中央値+40.7% / 平均+47.8%

### 3. 中間ビン（20-60%）：リターン低め＋不安定

- 平均+36〜41%、プラス確率70〜73%
- VTやSlopeMultが低い = 市場が不安定な過渡期
- **保有比率を下げる戦略判断が合理的**な局面

### 4. 10-20%ビン：特筆すべき安定性

- プラス確率80.2%で90-100%ビンを上回る
- 標準偏差64.2%で全ビン中最小クラス
- DD=1（HOLD）だがVTやSlopeが低い「慎重投資」局面 → 1年後のリターンは堅実

## 他投資先との比較用サマリー

raw_leverage が高い（90-100%）とき = 戦略が「投資すべき」と判断した平常時の期待値：

| 指標 | 3倍NASDAQ（本分析） | 参考：S&P500長期平均 | 参考：米国債券 |
|---|---|---|---|
| 平均年利 | +47.8% | +10〜12% | +3〜5% |
| 中央値 | +40.7% | +12〜14% | +3〜5% |
| プラス確率 | 78.1% | 約73% | 約85% |
| 最悪1年 | -97.5% | 約-38% | 約-13% |

> ※ S&P500・債券の値は一般的な長期実績の概算値。3倍NASDAQは日次リバランス・ボラティリティ減衰の影響を含む。

## 技術的補足

- **前方1年リターンの計算**: 各日tにおいて `Π(1 + 3×daily_return - daily_cost)` を t+1 〜 t+250 で算出
- **ウォームアップ除外**: MA200 + NORM_WINDOW(60) + Z_WINDOW(120) = 約380日 → 安全マージンとして400日を除外
- **raw_leverage分布**: 0-10%（21.3%）と90-100%（29.2%）に二極集中 → DD層のバイナリ特性が支配的
- **GASとの完全一致**: CONFIG値はすべて [NASDAQ-strategy-gas](https://github.com/KazuyaMurayama/NASDAQ-strategy-gas) の `Code.gs` と同一
