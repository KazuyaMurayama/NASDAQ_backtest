## §3 今セッションの主要発見 (5点)

### ① F8-R5「182 trades/yr」は計測バグだった → 実 52 回 (F10)

旧実装は `lev_mod`（LT2 バイアスで毎日連続変化する値）を取引カウント基準にしていた。
正しい基準は `lev_raw`（DH Dyn の離散出力）。修正後の実 Trades/yr = **52**（ユーザー許容 70 以内）。
F10 ε=0.015 は G7 WFA PASS (CI95_lo=+27.91%)。

### ② vz_thr=0.65 は過学習シグナル → 全凍結 (E5b + G6)

E5b で vz_thr=0.65 の山の形状を精査: 隣接値 (0.625/0.675) との Sharpe 差 **+0.043** → **スパイク確定**（閾値 0.03 超）。
G6 WFA で CI95_lo 優位は **+0.44pp のみ**（G7 F10 の +1.40pp の 1/3）。ノイズ域と判定。
vz065 系（単独・l_max=5.0 との組み合わせとも）全凍結。

### ③ l_max=5.0 は WFA PASS だが CI95_lo が E4 を下回る (G8)

E4+lmax5.0 の G8 WFA: CI95_lo=**+24.49%**（PASS、E4より −2.02pp）、WFE=+1.181。
F10+lmax5.0 の G8 WFA: CI95_lo=**+25.57%**（PASS、E4より −0.94pp）、WFE=**+1.278**（全候補最良）。
「CI95_lo で E4 超え」は F10 単独のみ。l_max=5.0 はコスト（CAGR −2.95pp）を払いつつ MaxDD 改善、Defensive 選択肢として有効。

1.5x コストストレス下で F10 の E4 比優位: +3.31pp → +3.30pp（ほぼ不変）。

### ⑤ マルチホライズン LT2 (N 合成) は効果なし → 棄却 (B10)

### F10 採用の根拠

- **統計的根拠 (機関投資家)**: WFA CI95_lo=+27.91% は E4 比 +1.40pp 上回り、vz065 の +0.44pp と比べ 3 倍以上の有意性。49 fold 平均での安定改善は「OOS 偶然」ではない。

- **過学習なし (H1 実測)**: 好調窓 10 件除去後も CI95_lo=+11.10% > 0。ストレス下でも優位保持。

### F10 採用の懸念点と対応策

  → **対応**: 四半期ごとに Worst10Y★ を監視。+16.0% 未満で警告。
- **MaxDD −63.09% (Actuary)**: E4 比 −3.08pp 悪化。
  → **対応**: ステップ 2 (G9) で F10+lmax5.0 検証。MaxDD を −57% 付近に抑えられるか確認。

### vz_thr=0.65 凍結の決定的根拠 (懐疑論者の指摘)
