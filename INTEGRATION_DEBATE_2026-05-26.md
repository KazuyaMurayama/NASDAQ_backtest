## §3 エージェント間議論

### Q1: F10 ε=0.015 を採用すべきか？

**Actuary**:
MaxDD が E4 比 -3.08pp 悪化 (-60.01% → -63.09%) は無視できない。年金資産的視点では「-63%」の最悪期は -60% よりも資金引出耐性で大差。さらに IS-OOS gap -4.31pp は警戒水準で、Worst10Y★ も -0.09pp 微減。ただし P10_5Y▷ は +0.5pp 改善している点は救い。総合的には **MaxDD 悪化を補う Sharpe +0.044 改善が真に持続するか**が論点で、WFA WFE=1.208 は β帯内なので構造的崩壊リスクは低い。**条件付き賛成**。

**Trader**:
Trades/yr 27→52 は実取引コストで -0.7〜-1.3pp/yr の追加負担 (R1C 試算)。Sharpe +0.044 改善はコスト後で +0.030 程度に縮む可能性。CAGR +3.31pp 改善はコスト後でも +2.0pp 前後残る計算なので、**ネット採用価値は維持される**。スプレッドは E4 と同じ DH Dyn 基底なので新規執行リスクなし。**賛成**。Trades/yr=52 はユーザー許容上限 70 内。

**Institutional Investor**:
WFA G7 で CI95_lo=+27.91% は E4 (+26.51%) を **+1.40pp 上回り、これは G6 vz=0.65 (+0.44pp) と比べて 3 倍以上の有意性**。49 fold 平均で安定して E4 を上回る点は機関投資家視点で最重要。Sharpe 改善が WFA で確認されたこと自体が「OOS 偶然」ではない強い証拠。WFE=1.208 も β帯中央で安定。**賛成**。

**Critical Thinker**:
IS-OOS gap -4.31pp の解釈に注意が必要。**IS<OOS は overfitting の通常徴候 (IS>OOS) とは逆方向**で、OOS 期間 (2021-2026 の 5 年) が偶然 IS より好調だった可能性を示す。WFA は 49 fold で IS↔post-IS の比較なので IS-OOS の片側性は希釈されている。**ただし**、F10 の tilt パラメータ (TILT=10.0, cap_eff=0.15/0.10/0.05) は F8-R5 段階での IS チューニング遺産で、これら 4 パラメータの汎化性は WFA の per-window 再フィッティングでは確認できない。CI95_lo +1.40pp は確かに有意だが、Sharpe +0.044 は GROSS であり、コスト後でも本当に E4 を上回り続けるかの試算が必要。**慎重賛成、ただし採用後の四半期モニタリング義務**。

**Hedge Fund Manager (集約)**:
4 視点ともに「採用すべき」方向。最重要根拠は **WFA G7 の CI95_lo=+27.91% が E4 比 +1.40pp 上回り、これは vz=0.65 単独の +0.44pp ノイズ域と比較して 3 倍の有意性**を示すこと。MaxDD -3.08pp 悪化は P10_5Y▷ +0.5pp 改善で部分相殺、Trades/yr=52 はコスト試算後でも CAGR ネット +2.0pp 改善見込み。**判断: F10 ε=0.015 採用推奨 (条件: Trades/yr ≤ 70 維持・四半期 Worst10Y★ モニタリング)**。

### Q2: l_max=5.0 を追加すべきか？

**Actuary**:
l_max=7.0 → 5.0 は MaxDD を -60.01% → -54.07% に **▲5.94pp 改善** (E4+lmax5.0)、Worst10Y★ は +18.7% → +16.9% で **-1.7pp 悪化** だが、P10_5Y▷ は **+9.8% → +12.1% で +2.3pp 改善**。年金引出耐性 (P10) の改善が Worst10Y の犠牲を上回る。CAGR -2.95pp 犠牲は実質金利環境では大きいが、5 年累計 -15% 程度の機会損失で、ドローダウン耐性を買う対価として妥当。**強く賛成 (Defensive Variant として)**。

**Trader**:
l_max は CFD 動的レバ上限のみ変更で、Trades/yr=27 不変・スプレッド負担も不変。**実取引コスト中立**で、信号品質変更なしのため運用継続性が高い。「7.0 で踏み込めない局面で 5.0 が天井に張り付く」ことによる強気相場時の取り逃しが懸念だが、年間取引数が変わらない以上は機会損失は CAGR 数値に既に反映されている。**賛成**。

**Institutional Investor**:
リスク調整後リターンで見ると、E4 (Sharpe 0.891, MaxDD 60%) と E4+lmax5 (Sharpe 0.896, MaxDD 54%) は **Sharpe ほぼ同等で MaxDD のみ改善**。Calmar 比 (CAGR/MaxDD) では E4=0.558, E4+lmax5=0.566 でわずかに改善。リスクパリティ目線では l_max=5.0 が優位。ただし **WFA 未実施が決定的に欠如**。G3/G6/G7 と同形式の WFA をかけて CI95_lo が >+26% を維持するかが採用条件。**条件付き賛成 (WFA 必須)**。

**Critical Thinker**:
l_max 削減は構造変更が単純 (CFD レバ上限のみ) で過学習しにくく、IS-OOS gap=-1.00pp は E4 (-1.81pp) より良好。**これは F10/vz=0.65 と本質的に異なるクリーンな改善**。ただし採用判断の論点は「CAGR -2.95pp を支払って MaxDD -5.94pp を買う価値はあるか」という主観的トレードオフで、**ユーザー効用関数次第**。客観的に過学習リスクは低い。**賛成、ただしユーザーのリスク選好確認必須**。

**Hedge Fund Manager (集約)**:
全員が改善方向は認める。論点は「採用形態」。
1. **メイン戦略を E4+lmax5 に変更**: CAGR 犠牲が大きい (-2.95pp/年 = 10 年で -25%) → 推奨せず
2. **Defensive Variant として併存**: CURRENT_BEST_STRATEGY に「Conservative variant」セクション追加、ユーザーがリスク選好で選択 → **推奨**
3. **F10 と l_max=5.0 の併用 (未測定)**: F10 ε=0.015 で CAGR ベースを引き上げつつ lmax=5.0 で MaxDD を抑える組み合わせ → **次ラウンド要実施**

**判断: l_max=5.0 単独採用は急がず、F10 採用後に「F10 + lmax=5.0」を検証してから Defensive Variant 化を判断**。

### Q3: vz_thr=0.65 は使えるか？

**Actuary**:
vz=0.65 + lmax=5.0 で MaxDD=-51.82% は **プロジェクト史上最良**。Worst10Y★ +18.33% も維持、P10_5Y▷ +11.87% は最良級。リスク指標は文句なし。ただし IS-OOS gap -3.91〜-5.22pp の OOS 偶然性疑いは深刻で、Worst10Y★ は IS+OOS 全期間ベースの指標なので「OOS 限定の幸運」を希釈している可能性が低いとはいえ、**vz_thr スパイク (隣接差 0.043) は WFA fold 別最適点のばらつきを強く示唆**。WFA 未実施の vz=0.65+lmax=5 の本格採用は時期尚早。**留保**。

**Trader**:
vz_thr の変更は signal generation のみ、Trades/yr=27 で実取引中立。執行リスクなし。問題は「vz_thr=0.65 と 0.70 のどちらが将来も最適か」の信頼性で、**G6 WFA で +0.44pp しか優位がない以上、実取引でこの差を確信して採用するのは難しい**。Sharpe 改善の大半はバックテスト偶然性に乗っている疑いが残る。**反対**。

**Institutional Investor**:
G6 WFA で CI95_lo=+26.95% は E4 (+26.51%) を **+0.44pp しか上回らない**。これは G7 F10 の +1.40pp と比較して **3.2 倍劣る**。CI95 の標準誤差を考えれば +0.44pp は明確にノイズ域。「Sharpe バックテスト +0.054 → WFA +0.008」のシュリンク率 85% は、E5b で確認されたスパイク形状と整合的で、**真の優位はほぼゼロ**と解釈すべき。**反対**。

**Critical Thinker**:
**最も鋭い指摘**: WFA は IS 期間 (1977-2021, 44年) のうちから 49 個の fold を切る方式だが、これらの fold の **post-IS 期間は全て IS 内部**である。つまり「OOS 期間 (2021-2026 の 5 年) 固有の vz 集中」に対するロバスト性は WFA でも完全には検証できていない。E5b で観測された IS-OOS gap -5.22pp は WFA で評価不能な OOS-specific 現象である可能性が高い。スパイク形状 (+0.0431) は **fold 別最適 vz_thr が 0.625〜0.675 に散らばる予兆**で、固定 vz=0.65 採用は将来 regime で破綻するリスクが高い。**強く反対**。

**Hedge Fund Manager (集約)**:
3/4 が反対、Actuary も留保。Critical Thinker の指摘「WFA は OOS 期間特異性を検証できない」は決定的。**判断: vz_thr=0.65 は単独でも併用 (vz=0.65 + lmax=5.0) でも採用見送り**。理由:
1. G6 WFA 優位 +0.44pp はノイズ範囲 (G7 F10 の +1.40pp と比較し 3.2 倍劣る)
2. IS-OOS gap -5.22pp + スパイク形状の二重シグナルは OOS 偶然性を強く示唆
3. CAGR/Sharpe 改善の 85% が WFA でシュリンクしている

**ただし**: vz=0.65 + lmax=5.0 の MaxDD=-51.82% は唯一無二の値。**Defensive Variant 候補としては将来再評価対象に残す**。

### Q4: 最優先の「次の一手」は何か？

**Actuary**:

**Trader**:
Trades/yr 増加が許容範囲 (52 ≤ 70) の F10 を即採用。CAGR +3.31pp はネットコスト後でも +2.0pp 残る計算で、E4+lmax5 の -2.95pp 犠牲より明確に優位。**推奨優先度: F10 採用 > lmax5 検証**。

**Institutional Investor**:
WFA エビデンスの強度で判断。G7 F10 (+1.40pp 有意) > G6 vz=0.65 (+0.44pp ノイズ) > lmax5 (WFA 未実施)。**F10 を即採用、次にF10+lmax5 を検証、最後に Defensive Variant 化**を判断。

**Critical Thinker**:
最も慎重な選択は **「F10 採用 + 同時に lmax=5.0 単独 WFA 並行実施」**。F10 は WFA 済みで証拠強度高、lmax=5.0 は構造シンプルで過学習しにくい。両方を別軸で進めることで、片方が失敗しても他方が残る冗長性。vz=0.65 系は完全凍結。
