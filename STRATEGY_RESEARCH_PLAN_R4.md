# 3倍レバレッジNASDAQ戦略 追加検証計画（Round 4）

> **目的:** 既存研究（R1〜R3, 255戦略）の成果を踏まえ、さらに有効な戦略を多角的に探索する
> **評価重視項目:** 年平均リターン（CAGR）、最悪5年平均リターン、シャープレシオ
> **取引回数制約:** 全期間100回以内（年2.1回以下）

---

## 1. 既存研究の分析と未探索領域

### 1.1 既存研究で確立された知見

| 発見 | 内容 | 最良結果 |
|------|------|----------|
| DD制御が基盤 | 200日最高値からの下落監視 | 取引29回、安定 |
| EWMA VTが最重要 | Span=10、Target Vol=25%が最適 | Sharpe +39%改善 |
| DD閾値最適化 | Exit -15%/-18%、Reentry 90%/92% | Sharpe 1.90達成 |
| デュアルDDが最低リスク | 100日+200日の2重監視 | MaxDD -30.56% |
| レジームフィルターは失格 | MA200クロスは取引200回超 | 実用不可 |

### 1.2 未探索・深掘り可能な領域

| 領域 | 既存研究での扱い | 探索余地 |
|------|------------------|----------|
| トレンド強度指標 | 未検証 | ADX相当、乖離率ベース |
| Vol-of-Vol | 軽く触れたのみ | 急変検出への活用 |
| ATR/ボリンジャー | 未検証 | 適応的閾値設定 |
| 季節性（低頻度） | 月別リターンのみ | 四半期、月末効果 |
| リスク調整モメンタム | 未検証 | ローリングSharpeベース |
| 適応的リエントリー | 固定閾値のみ | Vol状態で動的調整 |
| 複合DD（3タイムフレーム） | 2タイムフレームまで | 短期50日追加 |
| 非対称DD閾値 | 未検証 | 上昇相場と下落相場で別閾値 |

## 2. 検証戦略カテゴリ（10カテゴリ、計72戦略）

### カテゴリA：トレンド強度フィルター（12戦略）
**概念:** MA200クロスではなく、トレンドの「強度」を測定して投資判断

#### A1. 価格乖離率フィルター
- 価格がMA200からN%以上乖離している場合のみ投資
- MAクロスより低頻度（乖離が持続する間は取引なし）

| ID | 戦略 | パラメータ | 予想取引回数 |
|----|------|------------|--------------|
| A1-1 | DD(-18/92)+VT_E(25%)+乖離率フィルター | 乖離閾値: +5%以上で投資 | 40-50回 |
| A1-2 | DD(-18/92)+VT_E(25%)+乖離率フィルター | 乖離閾値: +3%以上で投資 | 50-60回 |
| A1-3 | DD(-18/92)+VT_E(25%)+乖離率フィルター | 乖離閾値: +0%以上で投資 | 60-70回 |

#### A2. ADX相当指標（方向性指数）
- 過去N日間の上昇日数比率でトレンド強度を判定
- 強トレンド時のみフルレバレッジ、弱トレンド時は縮小

| ID | 戦略 | パラメータ | 予想取引回数 |
|----|------|------------|--------------|
| A2-1 | DD(-18/92)+トレンド強度VT | 上昇日数比率>55%でフル、<45%で半減 | 35-45回 |
| A2-2 | DD(-18/92)+トレンド強度VT | 上昇日数比率>60%でフル、<40%で半減 | 30-40回 |
| A2-3 | DD(-18/92)+トレンド強度VT | LB=40日、閾値55%/45% | 40-50回 |

#### A3. モメンタムスコア（低頻度版）
- 過去60/120/200日のリターン加重平均でスコア化
- スコアが正の場合のみ投資（シグナル変化が低頻度）

| ID | 戦略 | パラメータ | 予想取引回数 |
|----|------|------------|--------------|

### カテゴリB：Vol-of-Vol（ボラティリティの不安定性）（8戦略）
**概念:** Volの変動が激しい時期は市場が不安定 → レバレッジ縮小

#### B1. Vol-of-Vol閾値制御
- 過去N日間のVol変動の標準偏差を計算
- Vol-of-Volが高い時期はレバレッジキャップを適用

| ID | 戦略 | パラメータ | 予想取引回数 |
|----|------|------------|--------------|
| B1-1 | DD(-18/92)+VT_E(25%)+VoV制御 | VoV>中央値1.5倍でLev上限2x | 29回（DD部分） |
| B1-2 | DD(-18/92)+VT_E(25%)+VoV制御 | VoV>中央値2.0倍でLev上限1.5x | 29回 |
| B1-3 | DD(-18/92)+VT_E(30%)+VoV制御 | VoV>中央値1.5倍でLev上限2x | 29回 |

#### B2. Vol急騰検出（Vol Spike）

- Vol急騰は暴落の先行指標となる可能性

| B2-3 | DD(-18/92)+VolSpike退避（VTなし） | Vol前日比>1.8倍で退避、7日後復帰 | 40-60回 |

## 4. 評価指標（詳細）

### 4.1 主要評価指標

| 指標 | 定義 | 重視度 |
|------|------|--------|
| **CAGR** | 年平均複利リターン | ★★★★★ |
| **最悪5年平均リターン** | 任意の連続5年間の最低平均リターン | ★★★★★ |
| **Sharpe Ratio** | (年率リターン - 0) / 年率Vol | ★★★★★ |
| **MaxDD** | 最大ドローダウン | ★★★★ |
| **Sortino Ratio** | 年率リターン / 下方偏差 | ★★★★ |
| **Calmar Ratio** | CAGR / |MaxDD| | ★★★ |
| **取引回数** | バイナリ状態遷移の回数 | ★★★（100回以内必須） |

### 4.2 追加評価指標（新規）

| 指標 | 定義 | 目的 |
|------|------|------|
| **最悪5年CAGR** | 連続5年間の最低CAGR | 長期低迷リスク評価 |
| **勝率（年次）** | 年次リターンが正の年の割合 | 安定性評価 |
| **回復期間** | MaxDDから回復までの日数 | 心理的耐性評価 |
| **危機年パフォーマンス** | 1974,1987,2000-02,2008,2020の個別成績 | ストレステスト |
