## A. 初回セットアップ（1回だけ）

> 以下の手順は **SBI証券 新アプリ版**（`member.c.sbisec.co.jp` またはSBIアプリ）を前提にしています。
> ナビゲーションが見当たらない場合は「外国株式トップ」→「取引」の順で目的の機能を探してください。

### A-1. SBI証券口座に日本円を入金

1. SBIアプリを開いてログイン（ID・パスワード ＋ 2段階認証）
2. 画面下部ナビバーまたは ≡ メニューから **「入出金・振替」** をタップ
3. **「入金」** をタップ → 振込元の金融機関を選択
   - 住信SBIネット銀行: 即時反映
   - 他行: 翌営業日以降
4. 金融機関アプリで振込を完了
5. **確認**: **「My資産」** または **「口座管理」** → **「口座サマリー」** → 「買付余力（円）」に入金額が反映されていること

### A-2. 円 → 米ドルに両替（為替手数料 0円）

3商品すべてがUSD建てのため、まとめてドルを準備します。

> **前提条件**: インターネットコース（プランC除く）であれば、リアルタイム為替取引の USD/JPY スプレッドが **無料**（SBI証券 ゼロ革命）。
> 「円貨決済」で注文すると「定時為替取引」扱いとなり **25銭/ドルのスプレッドが発生** するため、外貨決済を強く推奨。

**手順:**

1. SBIアプリ下部ナビまたは ≡ メニューから **「外国株式」** をタップ
2. 「外国株式トップ」画面で **「為替取引」** をタップ
   （見当たらない場合: **「取引」** タブ → **「為替取引」** または **「外貨両替」**）
3. 以下を入力:
   - 通貨ペア: **USD/JPY**
   - 取引区分: **買付**（円→ドル）
   - 注文方法: **リアルタイム（成行）**
   - 金額: TQQQ + TMF の購入予算合計をドル額または円額で入力（2036は円建てのため不要）
4. **取引パスワード** を入力 → **「注文確認」** → 内容確認 → **「注文発注」**
5. **確認**: **「口座管理」** → 「預り金（米ドル）」の残高に反映されていること
   （「外国株式」トップの「買付余力 USD」欄でも確認できます）

> **2回目以降**: 売却代金は自動的にUSDのまま口座に残ります。次回買付に再利用可能（再両替不要）。

### A-3. 初回取引前の同意確認

初めて米国ETFを購入するとき、以下の同意画面が表示されることがあります:
- 「外国証券取引口座約款への同意」
- 「外国証券取引に係る契約締結前交付書面の確認」

いずれも内容を確認の上、チェック → **「同意する」** で進んでください（1回のみ）。

## B. 毎回の運用フロー（GAS通知を受信したとき）

### B-1. GAS通知を確認し、売買株数を計算

GASスプレッドシートの最新行から `raw_leverage`・`w_nasdaq`・`w_gold`・`w_bond` を読み取り、目標株数を計算します。

#### 株数計算の公式

```
目標TQQQ額（円） = raw_leverage × w_nasdaq × 総資産（円）
目標2036額（円） = raw_leverage × w_gold   × 総資産（円）
目標TMF額（円）  = raw_leverage × w_bond   × 総資産（円）

TQQQ株数 = 目標TQQQ額 ÷ 為替レート ÷ TQQQ株価（USD）  ← 小数切り捨て
2036口数 = 目標2036額 ÷ 2036口価格（円）              ← 小数切り捨て
TMF株数  = 目標TMF額  ÷ 為替レート ÷ TMF株価（USD）   ← 小数切り捨て
```

> **初回購入は `raw_leverage` を使う**（`new_leverage` はGASが追跡中の既存ポジションベースのため）。
> 2回目以降のリバランスは `new_leverage` をGASが自動管理。

**例: 総資産 2,200万円、raw_leverage=0.4293、w_nasdaq=0.561、w_gold=0.22、w_bond=0.219、USD/JPY=159**

| 資産 | 配分 | 目標額（円） | 単価 | **目標数量** |
|------|------|------------|------|------------|
| TQQQ | 0.4293×56.1%=**24.1%** | 530万円 | $49 @ 159円 | **約681株** |
| 2036 | 0.4293×22.0%=**9.4%** | 207万円 | 202,000円 | **約10口** |
| TMF  | 0.4293×21.9%=**9.4%** | 207万円 | $36 @ 159円 | **約361株** |
| キャッシュ | 57.1% | 1,256万円 | — | — |

**差分（売買額）の計算:**

| 資産 | 現在保有 | 目標数量 | **売買** |
|------|---------|---------|---------|
| TQQQ | 0株 | 681株 | **681株 買付** |
| 2036 | 0口 | 10口 | **10口 買付** |
| TMF  | 200株 | 361株 | **161株 買付** |

### B-2. 売却を先に実行（資金確保）

ポジションを**減らす銘柄を先に売却**して、USD残高を確保してから買付を行います。

### B-3. 2036（Gold 2x）を東証で売買 【9:00〜15:30】

1. SBIアプリ → **「国内株式」**（または検索アイコン）
2. 銘柄検索欄に **`2036`** と入力 → 検索
3. **「NEXT NOTES 金先物ダブル・ブル ETN」** をタップ
4. **「現買」**（買付）または **「現売」**（売付）をタップ

**注文入力:**

| 項目 | 設定内容 |
|------|---------|
| 銘柄コード | `2036` |
| 数量 | 計算した口数（整数のみ・最低1口） |
| 価格 | **指値（推奨）** または 成行 |
| 指値の価格 | **前日終値 ± 1〜2%** 程度 |
| 期間 | 当日中 |
| 口座区分 | 一般 or 特定（ご自身の方針に合わせて） |
| 決済通貨 | **円**（FX変換不要） |

5. **取引パスワード** を入力 → **「注文確認」** → **「注文発注」**
6. **確認**: **「注文照会」** で当日中に **「約定済」** になることを確認

### B-4. TQQQ を米国市場で売買 【10:30〜】

1. SBIアプリ下部ナビまたは ≡ メニューから **「外国株式」** をタップ
2. 「外国株式トップ」の上部タブ **「取引」** をタップ
3. 「国名選択」が **「米国」** になっていることを確認
4. 銘柄検索欄に **`TQQQ`** と入力 → 検索
5. **「ProShares UltraPro QQQ（TQQQ）」** をタップ → **「現物買付」** または **「現物売付」**

**注文入力:**

| 項目 | 設定内容 |
|------|---------|
| 株数 | 計算した株数（整数のみ・最低1株） |
| 価格区分 | **指値（推奨）** または 成行 |
| 指値の価格 | **前日終値 ± 1〜2%**（例: $49 → 買付なら $50） |

| 口座区分 | 一般 or 特定 |
| 決済方法 | **外貨決済**（USD残高あり → 為替手数料0円） |

6. **取引パスワード** を入力 → **「注文確認」** → **「注文発注」**
7. **確認**: **「注文照会」** に表示されていること（約定は夜22:30以降）

## C. リバランス時の注意事項

1. **順序**: 売却（ポジション縮小銘柄）→ 買付（ポジション拡大銘柄）の順に実行
2. **USD残高管理**: 売却代金は自動的にUSDのまま外貨口座に残る。次回買付に再利用可能（両替不要）
3. **USD残高不足時**: 外貨決済で注文しても残高が不足すると、受渡日17:30に **自動で円→ドル両替（25銭/ドル手数料）** が発生する。事前にA-2の手順で補充すること
4. **米国休場日**: TQQQ・TMF は注文不可。**2036（東証）は米国休場に関わらず取引可能**。[SBI証券 米国休場日カレンダー](https://search.sbisec.co.jp/v2/popwin/info/stock/pop6040_ussec_closed.html) で確認
5. **東証休場日**: 2036 は東証休場日（祝日等）に注文不可。TQQQ・TMF は東証休場に関わらず取引可能
6. **確定申告**: TQQQ・TMF は米国ETF（配当は米国で10%源泉徴収 → 確定申告で**外国税額控除**申請可）。2036 は国内ETN（国内源泉徴収のみ）。一般口座の場合は3商品とも確定申告が必要
