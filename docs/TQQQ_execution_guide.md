## A. 初回セットアップ（1回だけ）

### A-1. SBI証券口座に日本円を入金

証券総合口座に投資資金がまだない場合のみ。

1. [SBI証券にログイン](https://site3.sbisec.co.jp/ETGate/) → 上部メニュー **「入金」** をクリック
2. 振込先の金融機関を選択（住信SBIネット銀行なら即時反映）
3. 金融機関サイトで振込を完了
4. 「口座管理」→「口座サマリー」で買付余力に反映されたことを確認

### A-2. 円 → 米ドルに両替（為替手数料 0円）

TQQQ購入用の米ドルを準備する。**外貨決済**を使うことで、注文ごとの為替コスト(25銭/ドル)を回避できる。

> **前提条件**: インターネットコース（プランC除く）の場合、リアルタイム為替取引の米ドル/円スプレッドが**無料**（[ゼロ革命第3弾](https://go.sbisec.co.jp/lp/zero_revolution_2023.html)）。
> **注意**: 「円貨決済」で米国株を注文すると「定時為替取引」扱いとなり**手数料無料の対象外**（25銭/ドル発生）。

1. [SBI証券にログイン](https://site3.sbisec.co.jp/ETGate/) → 上部メニュー **「外国株式 取引」** をクリック
2. 外国株式取引サイトが開く → 上部メニュー **「為替取引」** をクリック
3. 以下を入力:
   - 通貨: **USD/JPY**
   - 取引区分: **買付**（円→ドル）
   - 注文方法: **リアルタイム（成行）**
   - 買付金額: TQQQ投資額に相当する米ドル額 or 日本円額を入力
4. **取引パスワード** を入力 → **「注文確認画面へ」** → **「注文発注」**
5. 即時反映: 「口座管理」→ 外貨建口座の米ドル残高に反映

> 2回目以降: TQQQの売却代金は米ドルのまま保持されるので、**リバランスで売って買う場合は再両替不要**

### A-3. TQQQ初回買付時の確認事項

初めて米国ETFを買付する際、以下の確認画面が出る場合がある:
- **外国証券取引に係る契約締結前交付書面の確認・同意**
- 画面の指示に従いチェック → 「同意する」で進む（1回のみ）

## B. 通常の運用フロー（GAS通知受信後）

### タイムライン

```
Day 0  朝 7:00   GAS通知受信
                  「NASDAQ 3x: 68.0%, ゴールド: 17.6%, 債券: 14.4%」
Day 0  9:00-15:00 ① 東証で 447A, 2621 を売買 → 即約定
Day 0  10:30以降  ② TQQQ を指値で予約注文
Day 0  22:30(夏)  米国市場開場 → TQQQ約定
Day 1             TQQQ 国内約定日
Day 2             全商品 受渡完了 ✅
```

### B-1. GAS通知を確認し、売買額を計算

GAS LINEまたはメール通知の「ポートフォリオ配分」を確認。

**例: 総資産 1,000万円、現在 TQQQ 700万 / 447A 150万 / 2621 150万 の場合**

| 資産 | 通知の目標 | 目標額 | 現在保有 | **売買額** |
|------|-----------|--------|---------|----------|
| TQQQ | 68.0% | 680万 | 700万 | **売20万分** |
| 447A | 17.6% | 176万 | 150万 | **買26万分** |
| 2621 | 14.4% | 144万 | 150万 | **売6万分** |

**株数の計算:**
- TQQQ: 売買額(円) ÷ 為替レート ÷ TQQQ株価 = 株数
- 447A: 売買額(円) ÷ 447A口価格 = 口数
- 2621: 売買額(円) ÷ 2621口価格 = 口数

### B-2. 売却を先に実行（資金確保）

リバランスで**ポジションを減らす銘柄がある場合、買付の前に売却**を実行して資金を確保する。

### B-3. Gold (447A) を東証で売買 【9:00〜15:00】

1. [SBI証券にログイン](https://site3.sbisec.co.jp/ETGate/)
2. 画面上部の銘柄検索欄に **`447A`** と入力 → Enter
3. 銘柄ページが表示される → **「現物買」** or **「現物売」** をクリック
4. 注文入力:
   - **株数**: 計算した口数
   - **価格**: **成行** (推奨。流動性は十分) or 指値
   - **期間**: 当日中
   - **預かり区分**: **特定預かり**（確定申告の手間を減らす場合） or NISA（成長投資枠で節税する場合）
5. **取引パスワード** を入力 → **「注文確認画面へ」** → 内容確認 → **「注文発注」**
6. 約定確認: 上部メニュー **「注文照会」** で状態が **「約定済」** になったことを確認

### B-4. 米国長期債 (2621) を東証で売買 【9:00〜15:00】

1. 銘柄検索欄に **`2621`** と入力 → Enter
2. B-3と同じ手順で注文・約定確認

> **447Aと2621は同時に注文可能**（東証の別銘柄なので並行OK）

### B-5. TQQQ を米国市場で売買 【10:30〜翌朝】

#### 注文受付時間

| 時期 | 注文可能時間(日本時間) | 市場開場(日本時間) |
|------|----------------------|-------------------|
| 夏時間(3月〜11月) | 10:30頃 〜 翌5:00 | 22:30 〜 翌5:00 |
| 冬時間(11月〜3月) | 10:30頃 〜 翌6:00 | 23:30 〜 翌6:00 |

> 市場開場前に出した注文は、開場と同時にマッチング

#### 手順

1. [SBI証券にログイン](https://site3.sbisec.co.jp/ETGate/) → 上部メニュー **「外国株式 取引」** をクリック
2. 外国株式取引サイトが開く → 上部メニュー **「取引」** → **「米国」** を選択
3. 注文入力画面で以下を設定:
   - **ティッカー**: **`TQQQ`** と入力
   - **取引種類**: **現物買付** or **現物売付**
   - **株数**: 計算した株数
   - **価格**:
     - **指値（推奨）**: 前日終値 ± 1〜2% 程度に設定。米国市場にはストップ高/安がないため、成行は想定外の価格で約定するリスクあり
     - **成行**: 確実に約定させたい場合。ただし買付余力を多めに拘束される
   - **期間**: **当日中**
   - **預かり区分**: **特定預かり**（TQQQはNISA対象外）
   - **決済方法**: **外貨決済**（A-2で米ドル準備済みの場合）
     - 米ドル残高が不足の場合は **円貨決済** も可（ただし為替手数料25銭/ドル発生）
4. **取引パスワード** を入力 → **「注文確認画面へ」** → 内容確認 → **「注文発注」**
5. 約定確認（翌朝）: 外国株式取引サイト → **「注文照会」** で **「約定済」** を確認
   - または: メインサイト → **「口座管理」** → **「取引履歴」** → **「外貨建口座」**

#### 注文タイミングの推奨

| タイミング | メリット | デメリット |
|-----------|---------|-----------|
| Day 0 日中に予約注文(指値) | 通知後すぐ対応完了。夜起きなくてよい | 指値が離れすぎると約定しない |
| Day 0 夜22:30以降に成行 | 確実に約定 | 夜起きている必要あり |
| **Day 0 昼間に指値（前日終値±1%）**（推奨） | 1回の操作で完了。翌朝約定確認のみ | 極端な急変時は約定しない可能性 |

## C. リバランス時の注意事項

1. **売り→買いの順序**: ポジション縮小銘柄を先に売却 → 資金確保 → 拡大銘柄を買付
2. **TQQQ売却代金の再利用**: 売却代金は米ドルのまま外貨口座に残る → 次回買付にそのまま使える（再両替不要）
3. **米ドル残高不足時**: 外貨決済で注文しても米ドル残高不足の場合、国内受渡日17:30までに不足を解消しないと、**自動的に円→ドル両替（25銭/ドル手数料）** が実行される
4. **米国祝日**: 米国市場休場日はTQQQの注文不可。[SBI証券 米国休場日カレンダー](https://search.sbisec.co.jp/v2/popwin/info/stock/pop6040_ussec_closed.html) で事前確認
5. **確定申告**: TQQQ(米国ETF)の配当には米国で10%源泉徴収 → 確定申告で**外国税額控除**を申請可能。特定口座(源泉あり)なら譲渡益の申告は不要
6. **447A/2621がNISA口座の場合**: NISA口座での売却は非課税だが、損失は損益通算できない点に注意

## G. 検証方法

1. GAS `dryRun()` を実行し、ポートフォリオ配分が正しく表示されることを確認
2. `testNotification()` で更新後の通知フォーマットを確認
3. SBI証券で上記リンクからTQQQ/447A/2621の画面遷移が手順通りに動作することを手動確認
4. 少額(1株/1口)で実際に売買テストし、約定→受渡の日数を実測
