## 3. 取引タイムライン

3資産すべてが米国市場での取引となるため、**すべて同じ夜間枠でまとめて処理**できます。

```
Day 0  朝 7:00     GAS通知受信
                    「NASDAQ 3x: 68%, Gold 2x: 16%, 債券 3x: 16%」
Day 0  10:30〜     SBI証券で TQQQ・UGL・TMF の3銘柄を予約注文（指値）
Day 0  22:30(夏)   米国市場開場 → 3銘柄が一斉に約定
       23:30(冬)
Day 1              3銘柄 国内約定日
Day 2              3銘柄 受渡完了 ✅
```

> 市場開場前（日本時間10:30〜22:30）に指値で予約注文しておけば、夜に起きて操作する必要はありません。

## A. 初回セットアップ（1回だけ）

### A-1. SBI証券口座に日本円を入金

1. [SBI証券 ログイン](https://site3.sbisec.co.jp/ETGate/) → 画面上部メニュー **「入金」** をクリック
2. 振込先の金融機関を選択（住信SBIネット銀行なら即時反映）
3. 金融機関サイトで振込を完了
4. **確認**: 上部メニュー **「口座管理」** → **「口座サマリー」** → 買付余力に反映済みであること

### A-2. 円 → 米ドルに両替（為替手数料 0円）

3商品すべてがUSD建てのため、まとめてドルを準備します。

> **前提条件**: インターネットコース（プランC除く）であれば、リアルタイム為替取引の USD/JPY スプレッドが **無料**（SBI証券 ゼロ革命）。
> 「円貨決済」で注文すると「定時為替取引」扱いとなり **25銭/ドルのスプレッドが発生** するため、外貨決済を強く推奨。

**手順:**

1. [SBI証券 ログイン](https://site3.sbisec.co.jp/ETGate/)
2. 上部メニュー **「外国株式 取引」** をクリック
   → 外国株式取引サイト（別画面）が開きます
3. 外国株式取引サイトの上部メニューで **「為替取引」** をクリック
4. 以下を入力:
   - 通貨: **USD/JPY**
   - 取引区分: **買付**（円→ドル）
   - 注文方法: **リアルタイム（成行）**
   - 金額: TQQQ + UGL + TMF 購入予算の合計に相当するドル額（または円額）
5. **取引パスワード** を入力 → **「注文確認画面へ」** → 内容確認 → **「注文発注」**
6. **確認**: 外国株式取引サイト → **「口座管理」** → 外貨建口座の **「米ドル残高」** に反映済みであること

> **2回目以降**: 売却代金は米ドルのまま口座に残ります。リバランスで売って買う場合は再両替不要。

### A-3. 初回取引前の同意確認

初めて米国ETFを購入するとき、以下の同意画面が表示されることがあります:
- 「外国証券取引口座約款への同意」
- 「外国証券取引に係る契約締結前交付書面の確認」

いずれも内容を確認の上、チェック → **「同意する」** で進んでください（1回のみ）。

## B. 毎回の運用フロー（GAS通知を受信したとき）

### B-1. GAS通知を確認し、売買株数を計算

GAS通知（LINE または メール）に記載された目標配分と、現在のポジションを比較します。

**例: 総資産 1,000万円（≒ 62,893 USD @ 159円）、通知の目標配分が 68% / 16% / 16% の場合**

| 資産 | 目標% | 目標額（円） | 目標額（USD） | 株価（USD） | **目標株数** |
|------|-------|------------|-------------|-----------|------------|
| TQQQ | 68% | 680万円 | 42,767 USD | $49 | **873株** |
| UGL  | 16% | 160万円 | 10,063 USD | $63 | **159株** |
| TMF  | 16% | 160万円 | 10,063 USD | $36 | **279株** |

> **株数計算**: `目標額（円） ÷ 為替レート ÷ 株価` を小数点以下**切り捨て**

**差分（売買額）の計算:**

| 資産 | 現在保有 | 目標株数 | **売買** |
|------|---------|---------|---------|
| TQQQ | 900株 | 873株 | **27株 売却** |
| UGL  | 140株 | 159株 | **19株 買付** |
| TMF  | 270株 | 279株 | **9株 買付** |

### B-2. 売却を先に実行（資金確保）

ポジションを**減らす銘柄を先に売却**して、USD残高を確保してから買付を行います。

### B-3. 米国株取引画面へのアクセス

1. [SBI証券 ログイン](https://site3.sbisec.co.jp/ETGate/)
2. 上部メニュー **「外国株式 取引」** をクリック
   → 外国株式取引サイト（別画面）が開きます
3. 外国株式取引サイトの上部メニュー **「取引」** → **「米国株式」** をクリック

以降の手順は TQQQ / UGL / TMF で共通です（ティッカーを変えるだけ）。

### B-4. TQQQ の売買

**銘柄の検索:**

1. B-3 の画面で、銘柄検索欄に **`TQQQ`** と入力 → **「検索」** または Enter キー
2. 検索結果から **「ProShares UltraPro QQQ」** をクリック
3. 銘柄ページで **「現物買付」** または **「現物売付」** ボタンをクリック

**注文入力:**

| 項目 | 設定内容 |
|------|---------|
| ティッカー | `TQQQ` （自動入力済み） |
| 取引区分 | 現物買付 / 現物売付 |
| 株数 | 計算した株数を入力（整数のみ） |
| 価格区分 | **指値（推奨）** または 成行 |
| 指値の価格 | **前日終値 ± 1〜2%** 程度（例: 前日終値$49 → 買付なら$50程度） |
| 期間 | 当日中 |
| 口座区分 | **特定預かり** |
| 決済方法 | **外貨決済**（USD残高がある場合、為替手数料0円） |

> **指値 vs 成行:**
> - **指値（推奨）**: 前日終値に近い価格を設定。日中（10:30以降）に予約し、夜の開場を待つだけ。
> - **成行**: 必ず約定するが、急騰・急落時に予想外の価格で約定するリスクあり。米国株に値幅制限はありません。

4. **取引パスワード** を入力 → **「注文確認画面へ」** → 内容確認 → **「注文発注」**
5. **確認**: 上部メニュー **「注文照会」** → 注文一覧に表示されていることを確認

### B-5. UGL の売買

B-4 と同じ手順で、ティッカーを **`UGL`** に変更して実行。

| 項目 | 設定内容 |
|------|---------|
| ティッカー | `UGL` |
| 取引所 | NYSE Arca（SBI証券では「NYSE」として表示） |
| その他 | B-4 と同様 |

### B-6. TMF の売買

B-4 と同じ手順で、ティッカーを **`TMF`** に変更して実行。

| 項目 | 設定内容 |
|------|---------|
| ティッカー | `TMF` |
| 取引所 | NYSE Arca（SBI証券では「NYSE」として表示） |
| その他 | B-4 と同様 |

> **3銘柄まとめて発注可能**: TQQQ → UGL → TMF の順に注文画面を開いて続けて入力できます。いずれも同じ夜の米国市場で約定します。

### B-7. 約定確認（翌朝）

1. [SBI証券 ログイン](https://site3.sbisec.co.jp/ETGate/) → **「外国株式 取引」**
2. 外国株式取引サイト → **「注文照会」** → 3銘柄すべてが **「約定済」** になっていることを確認
3. または: メインサイト → **「口座管理」** → **「保有証券一覧」** → 外貨建口座 → 株数が更新されていることを確認

## C. リバランス時の注意事項

1. **順序**: 売却（ポジション縮小銘柄）→ 買付（ポジション拡大銘柄）の順に実行
2. **USD残高管理**: 売却代金は自動的にUSDのまま外貨口座に残る。次回買付に再利用可能（両替不要）
3. **USD残高不足時**: 外貨決済で注文しても残高が不足すると、受渡日17:30に **自動で円→ドル両替（25銭/ドル手数料）** が発生する。事前にA-2の手順で補充すること
4. **米国休場日**: TQQQ・UGL・TMFすべて注文不可。[SBI証券 米国休場日カレンダー](https://search.sbisec.co.jp/v2/popwin/info/stock/pop6040_ussec_closed.html) で確認
5. **確定申告**: 3商品とも米国ETF。配当は米国で10%源泉徴収 → 確定申告で**外国税額控除**申請可。特定口座（源泉あり）なら譲渡益の申告不要
