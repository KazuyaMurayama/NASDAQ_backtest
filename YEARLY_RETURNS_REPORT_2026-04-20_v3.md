## 注意事項

1. **レバレッジの違い**: BRK は実質 1x の株式会社運営。DH [A] は **TQQQ 3倍レバ + Gold 2x + Bond 3x** のスリーブ独立型。同CAGR比較はリスクを考慮した解釈が必要。
2. **MaxDD 比較**: DH [A] = -32.55% (2022年近辺)、BRK = 推定 ~-50% 前後 (1974/2008の大幅下落)。**DH [A] はリターンもリスクもBRKを上回る結果**(ただしバックテスト前提の限界は残る)。
3. **BRK の最強期は1965-1973**(CAGR +37%/年)、本レポートは1974開始のため含まれない。この期間込みで比較するとBRK優位年は増える可能性。
4. **2025年以降の BRK データ**は本バージョンに反映されない(公式年次開示が次期Annual Letterで更新される)。
5. **NASDAQ は配当込みか**: 本レポートは Total Return ベースで NASDAQ Composite を使用。BRK 配当無配当なので調整不要。
