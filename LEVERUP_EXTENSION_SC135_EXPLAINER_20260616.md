# 戦略解説 — #1 scale 1.35（強map）と P09 の違い

作成日: 2026-06-16
最終更新日: 2026-06-16

> **目的**: CAGR押上げ検証（[LEVERUP_EXTENSION_RESULTS_20260616.md](LEVERUP_EXTENSION_RESULTS_20260616.md)）で「CAGR上端の現実解」と整理された **#1 scale 1.35（強map）** を、平易に解説する。シグナル・扱う商品・おおよその配分、および [P09（P09_C1）](CURRENT_BEST_STRATEGY.md) との違いを示す。
> 結論を先に: **両者は同じ戦略骨格・同じシグナル・同じ商品。違いは「IN（強気）の日にどれだけレバレッジをかけるか」というダイヤル設定だけ**。

---

## 1. 戦略の全体像（P09・scale1.35 共通の骨格）

「3つのシグナルで、3つの局面を切り替える」構造。

| # | シグナル | 役割 |
|---|---|---|
| 1 | **DH-W1**（タイミング判断・非対称ヒステリシス） | NASDAQに乗る「IN」か守りに退く「OUT」かを決める状態スイッチ。**Enter≥0.7でIN入り・Exit≤0.3でOUT退避**（一度入ると簡単には出ない＝ダマシ低減）。**全営業日の約53%がIN・約47%がOUT** |
| 2 | **mom63ブースト**（IN日のレバ倍率） | IN日に、NASDAQ63日モメンタムの四分位でレバ倍率を決める。**弱モメンタム（Q0＝押し目）ほど強くレバを掛ける逆張り設計**（フォワード検証で「弱mom×高レバ」局面の先5日が年率+121%と最強だったため） |
| 3 | **bond_mom252ゲート**（OUT日の中身） | OUT日に、債券モメンタムが正なら債券を持つ・負ならその分を現金（SOFR利回り、＝C1）にする |

> Q0〜Q3 は mom63 の四分位。Q0＝直近モメンタム最弱（最も売られた状態）、Q3＝最強。ブーストは**逆張り**で、弱いほど強く張る。

## 2. 扱う商品（P09・scale1.35 共通）

| 局面 | 商品 | 役割 |
|---|---|---|
| IN（攻め） | **TQQQ**（3倍NASDAQ-100 ETF） | 3倍までのレバ部分（最安） |
| IN（攻め・3倍超のみ） | **くりっく株365 NASDAQ-100**（取引所CFD） | 3倍を超える超過分。店頭CFDより金利が安い（SOFR+0.75pp） |
| OUT（守り） | **Gold 1倍**（GLD相当） | OUT日は常に保有 |
| OUT（守り） | **米国債20年 1倍**（TLT相当） | OUT日に債券モメンタムが正のとき |
| OUT（守り） | **現金（SOFR利回り）** | OUT日に債券モメンタムが負のとき |

## 3. P09 との違い ＝「レバの強さ」だけ

| 項目 | P09（P09_C1） | **#1 scale 1.35 強map** |
|---|---|---|
| シグナル（DH-W1・mom63・bond_mom252） | — | **完全に同一** |
| 扱う商品（TQQQ/くりっく株365/金/債/現金） | — | **完全に同一** |
| OUT充填（金常時＋債タイミング＋C1現金利回り） | — | **同一** |
| mom63ブーストマップ {Q0,Q1,Q2,Q3} | {1.20, 1.10, 1.00, 1.00}（控えめ） | **{1.60, 1.50, 1.10, 1.00}（強い）** |
| 一律レバ倍率 | ×1.0 | **×1.35** |
| → 実効レバが3倍を超える日 | 約6% | **約43%** |
| くりっく株365の使用 | ほぼ不要 | **常用**（>3倍日が43%あるため） |
| min(IS,OOS) CAGR（税後⓽） | +17.77% | **+23.83%** |
| MaxDDⓒ | −35.0% | −45.0% |
| Sharpeⓒ | 0.912 | 0.882 |

**要するに**: P09は「弱モメンタム日に少しだけ（1.2倍）増やす・全体は等倍」の控えめ運用。scale1.35強mapは「弱モメンタム日に大きく（1.6倍）増やす・さらに全体を1.35倍」する強気運用。**同じ判断ロジックで踏み込む量を上げただけ**。結果、CAGRが+6pp高く、MaxDDも約10pp深く、3倍超のためにくりっく株365を常時使う。

## 4. おおよその配分（商品ごと＋キャッシュ）

時間（営業日）ベースの概算（OUT約47%・うち債券ON/OFFがほぼ半々・金/債は逆ボラ加重〔債券は低ボラなので多め〕という設計からの推定）。

| 資産 | 時間配分（概算） | 備考 |
|---|---:|---|
| **NASDAQ（TQQQ＋くりっく株365）** | **約53%の日** | IN日。**ただしレバ込み**で金額エクスポージャーは資本の3〜5倍規模（弱mom日に強ブースト） |
| **Gold 1倍** | 約17% | OUT日に常に保有（逆ボラで約35-40%枠 × OUT47%） |
| **米国債20年 1倍** | 約15% | OUT日∧債券モメンタム正のとき |
| **現金（SOFR利回り）** | 約15% | OUT日∧債券モメンタム負のとき |

> ⚠ **重要**: 上は「日数×枠」の概算。**IN日のNASDAQはレバレッジがかかっている**ため、ドル建てのリスク量は53%どころか資本の数倍になる（これが高CAGRと深いMaxDD−45%の源泉）。3倍まではTQQQ、3倍を超える約43%の日の超過分がくりっく株365。正確な平均ウェイト（各資産の時間平均比率・IN日の平均実効レバ・>3x超過量）はNAV/ウェイト系列から実値集計が可能（未実施）。

## 5. 採用上の注意（QC由来・正直な留保）

- scale1.35強mapは**「P09より良い戦略」ではなく「同じ戦略の強気版（リスク選好の選択肢）」**。**Sharpe（リスク調整後）はむしろP09より低く**（0.882 vs 0.912）、CAGR増はDD増との交換（純レバアップ）。
- 独立4エージェントQC（[LEVERUP_EXTENSION_QC_SIGNOFF_20260616.md](LEVERUP_EXTENSION_QC_SIGNOFF_20260616.md)）: 数値は正しく、**CAGR優位はOOS境界(2019-2023)に頑健**（単一強気OOS依存ではない）。ただし bear相場では比例超で悪化（trend:bear −5.40%）、Worst5Y は辛うじて+0.33%、worst暦年 −26%。
- 未モデルの限界: 追証/強制ロスカット・容量(くりっく株365建玉上限)・実運用MaxDDギャップ（バックテスト−45%は実運用でさらに深い可能性）。高レバ採用時はこれらの実務検証が必要。
