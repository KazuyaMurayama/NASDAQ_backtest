**観察**:
- 実 gap = -1.27pp は NULL 分布の下位 7% (5% 閾値を僅か上回る = marginal)
- 実 CAGR_OOS = +21.49% は NULL 上位 7% (同上)
- NULL 分布の mean = +1.09pp → NULL では gap 正方向、実 gap が逆方向で珍しい

**結論**: **marginal** — 5% 水準では足りないが、10% 水準では有意。
