## 5. 解釈と判断含意（率直に）

1. **あなたの仮説「指標を変えると CFD が魅力的になるか？」→ 答えは No、むしろ逆。**
   vz065_l5 は min(IS,OOS) では +16.84% と見栄えするが、**CPCV p10 +6.67%（最下位）・Regime_min −14.89%（最悪）・全ストレス窓マイナス・WFE1.348（regime luck旗）**。新評価系は**vz065_l5の脆弱性を露出**させる。CFDを残した判断は「選択肢保持」としては妥当だが、**この指標群で見ると l5 は採用根拠が弱まる**。
   - **補足（min と WFE の見かけの不一致）**: l5 は単一split OOS が +20.85% > IS +16.84% で「OOSが良い」ように見えるが、**WFE1.348 はローリング49窓平均で IS≫postIS（後半窓で大きく劣化）**を示す。つまり「たまたま1本のOOS窓が良かった」典型例で、**単一split評価がl5を過大評価していた**ことを WFA/CPCV が暴いている（本プロジェクトの主旨そのもの）。
2. **P09_TQQQ / LU1 が両ヘッドライン＋頑健性で一貫して上位。** 残る留保は (a) baseline比の平均エッジが bootstrap非有意、(b) 2015型（三資産同時安）の −18.7%。
3. **min と WFA CI95 は順位一致**＝当面 co-primary 併記で実害なし。主従を急いで決める必要はない（あなたの「結果を見て決める」は妥当）。
4. **MaxDD許容で CAGR を採るなら LU1 > P09**。ただし LU1 は29%が実効>3×でCFD/信用要（コスト計上済）。**CFD依存を避けたいなら P09_TQQQ が最適点**（ほぼ全期≤3×・TQQQ実現可・Regime_min最良 −1.57%）。

## 6. 昇格判定：バランス型スコアカード（return頑健性だけで決めない）

> 注: 「return下限を4通りに切った基準」は過学習防止には効くが MaxDD/Sharpe/テール/コストを欠く＝**単独では不適**。下記6次元で見る（[計画 §3.2](EVALUATION_METHODOLOGY_UPGRADE_PLAN_20260611.md) 参照）。**P09_TQQQ を baseline(V7-TQQQ) と全次元で比較**:

| 次元 | 指標 | V7基準 | P09_TQQQ | 判定 |
|---|---|---:|---:|---|
| ①リターン水準 | min / WFA CI95 | +16.27% / +15.48% | +17.59% / +17.94% | ✅ 両方上 |
| ②頑健性 | CPCV p10 / Regime_min / WFE | +10.37% / −6.14% / 0.98 | +13.56% / **−1.57%** / 1.02 | ✅ 大幅上 |
| ③リスク | MaxDD / Sharpe | −34.47% / 0.877 | −35.18% / **0.901** | △ DD −0.7pp / Sharpe↑ |
| ④テール | Worst10Y★ / P10 | +10.08% / +5.15% | **+11.45% / +6.56%** | ✅ 上 |
| ⑤コスト | Trades / >3×比率 | 25.2 / ~0% | 29.2 / ~0%(TQQQ実現可) | △ +4回 |
| ⑥有意性 | bootstrap / permutation | — | P=0.80(平均非有意) / permPASS | △ 平均未確証 |

- **P09_TQQQ は ①②④で明確上位、③⑤は僅差トレードオフ、⑥のみ平均エッジ未確証**。＝「一部基準特化」ではなく**多次元で総合的に baseline 以上**。CAGR優先・DD許容のあなたの選好では、③の −0.7pp DD と⑤の+4回は許容範囲。**Active候補への昇格は多次元基準でも支持される**（最終はユーザー承認）。
- **LU1**: P09 と同傾向でCAGRさらに上(+18.12%)だが ⑤で Trades35＋29%>3×(CFD依存)。攻め優先ならLU1、実装単純さ重視ならP09。
- **vz065_l5**: ②で全滅（CPCV p10最下位・Regime_min −14.89%）＝**return頑健性の時点で不合格**。残置は選択肢として可だが昇格根拠なし。
