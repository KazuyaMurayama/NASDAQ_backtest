## 2. 分析基準：SOFR Financing Dragとは何か

### 2.1 なぜSOFRがCAGRを-8.13%も引き下げるのか

TQQQ（3倍レバレッジETF）は「自己資金1 + 借入2 = 運用3」の構造で運用される。借入コストは米国翌日物担保調達金利（SOFR）に連動するため：

```
TQQQ年間financing cost = 2 × SOFR_USD
```

**期間別SOFR推移とドラッグ推計：**

| 期間 | 平均SOFR | 年間financing drag（2xSOFR） |
|---|---|---|
| 2010–2021（低金利） | ≈0.4% | ≈0.8%/yr |
| 2022–2023（利上げ） | ≈3.5% | ≈7.0%/yr |
| 2024 | ≈5.0% | ≈10.0%/yr |
| 2025 | ≈4.5% | ≈9.0%/yr |
| **2026-05現在** | **3.60%** | **7.2%/yr** |

バックテスト（1974–2026, 52年平均）での -8.13% CAGR はこの長期平均ドラッグを反映。  
**現環境（3.60%）ではドラッグは年7.2%。高金利が継続する限り圧迫は続く。**

### 2.2 JPYヘッジ商品のSOFRドラッグ相殺メカニズム

JPYヘッジ型の国内投資信託（例：NASDAQ100 3倍ブル）がNASDAQ100先物を使う場合：

1. 先物買建て → USD financing cost が先物プレミアムに内包（≈2×SOFR_USD）
2. USD→JPY ヘッジ（為替先渡売り）→ USD金利を得てJPY金利を支払う（≈-(SOFR_USD - BOJ_rate)）

**ネット実効financing cost = 2 × BOJ_rate**

```
JPYヘッジ3x fund financing cost = 2 × 0.75% = 1.50%/yr（現在）
vs TQQQ: 2 × 3.60% = 7.20%/yr

差額: +5.70%/yr の優位性（JPYヘッジ側）
```

## 8. 戦略的考察と推奨

### 8.1 核心的論点

現在のSOFR-BOJ金利差（+2.85%pt）により、**JPYヘッジ付き3倍商品はTQQQより年約6%低コスト**。  
しかし DH Dyn 2x3x 戦略はタイミングが命であり、**5営業日の遅延は月次シグナルを実質的に無効化しかねない**。

### 8.2 シナリオ別推奨

**シナリオA: タイミング精度を最優先（現行戦略維持）**
→ **TQQQ継続**。コストは高いが戦略の設計通りに動く唯一の選択肢。  
高金利が続く限り損だが、低金利局面（SOFR < 2%）では逆転する。

**シナリオB: コスト削減を最優先（戦略を修正してでも）**
→ **NASDAQ100 3倍ブル**に移行し、戦略の「DELAY」パラメータを5〜7に修正してバックテスト再評価。  
年6%のコスト差は大きく、シグナル遅延のペナルティを超える可能性がある。

**シナリオC: 2倍で妥協する場合**
→ **iFreeETF 2869（2倍）**。タイミング適合・コスト低い。ただし戦略の3倍前提を変更する必要あり。

**シナリオD: CFDは非推奨**
→ 長期保有・月次戦略では金利コストが高く、税務も複雑。採用しない。

### 8.3 未解決の重要論点（追加バックテスト推奨）

1. **DELAY=5〜7 でのNASDAQ100 3倍ブル代替バックテスト** → 遅延ペナルティの定量評価
2. **JPYヘッジ付き3倍商品の過去リターンデータ収集** → SOFR効果の実証
3. **SOFR金利見通し** → Fedが0%まで戻れば両者の差は縮む（シナリオA有利化）
4. **SBI証券（メイン口座）でのNASDAQ100 3倍ブル取扱確認** → SBIネオトレードは確認済みだが本口座は要確認
