## 1. 結論（先に要点）

1. **A1: 高頻度CFD系4戦略は全て CLOSE（正式クローズ）**。realistic コスト（時変SOFR+3.0%×フルNotional＋売買スプレッド）での after-tax IS CAGR は +16.58〜+16.81% で、閾値 18% に全戦略届かない。さらに MaxDD −67〜−69% でハードベト圏（−50%超）でもある。計画 §0.1 の予想（「残らない」）どおり。
2. **P01_w63 も正式 CLOSE**。min(IS,OOS)⓽ +14.34%（OOS⓽ +14.34%）で①リターン水準が決定的に不足。IS-OOS gap +4.81pp は過学習の兆候そのもの（重みモメンタム窓 w63 のIS適合）。6次元フルゲートに回す価値なし。
3. **「realistic コスト下で IS⓽ 20%超なのに不採用だった戦略は LU2 のみ」が確定**。LU2 の6次元フルゲート評価（B1）と、その発展形（B3レバ強化・C1/C4コスト改善）は Phase B/C レポートで扱う — **そこで min⓽ 20%超・ベト無しの候補が出ている**。

## 2. A1: realistic 再計算の結果

### 2.1 実装忠実性の証明（scenarioD 照合）

各戦略を旧 scenarioD コスト（CFD_SPREAD_LOW=0.20%/yr）でも走らせ、記録済みの旧数値と照合してから realistic を算出した:

| 戦略 | 照合指標 | 計算値 | 記録値 | 判定 |
|---|---|---:|---:|---|
| F10ε=0.015 | CAGR_OOS / Sharpe / Trades | +36.84% / 0.935 / 51.6 | +36.80% / 0.934 / 52 | ✅ |
| F8-R5 | Sharpe / Trades | 0.934 / 181.6 | 0.934 / 182 | ✅ |
| F7v3+E4 | Sharpe / Trades | 0.926 / 183.1 | 0.926 / 183 | ✅ |
| vz065_l7+F10ε | CAGR_OOS | +31.64% | +31.60% | ✅ |
| E4（参照） | IS⓽ realistic | +16.93% | +16.93% | ✅ |

実装所在: F10ε/F8-R5 = `src/g14_wfa_sbi_cfd.py` + `src/g19a_f10_eps_extended.py`、F7v3 = `src/f7v3_bull_tilt.py`、vz065_l7 = `src/audit/strategy_runners.py: run_vz065(7.0, basis)`。

**読み方**:
- 旧 scenarioD の「OOS 30%超」は誤コスト（低スプレッド0.20%/yr）の産物で、realistic 是正後の IS⓽ は E4 とほぼ同水準（16.6〜16.8%）に収斂する。ε デッドバンドや tilt の違いは IS⓽ では 0.2pp 程度しか効かない。
- 仮に IS⓽ が 18% を超えていても、MaxDD −67〜−69% は6次元評価のハードベト（−50%）で失格になるため、いずれにせよ昇格余地はなかった。
- 副産物の是正: MEMORY 等の「vz065_l7 OOS +22.5%」は realistic 値（+22.47%）であり、scenarioD 値（+31.6%）との混同があった。本再計算で系譜を確定。

## 4. Phase A の総括

- 「OOS偏重で落とした IS高CAGR戦略の再発掘」という仮説検証は、**LU2 以外は全滅という形で白黒がついた**（安価に確定、計画どおり）。
- CAGR 20%+ への道は「過去戦略の発掘」ではなく「**LU2系のレバ強化×コスト改善**」にある。続きは Phase B/C レポートへ。
