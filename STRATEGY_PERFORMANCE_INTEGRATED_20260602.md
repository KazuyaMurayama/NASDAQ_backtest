### §7-1 日次取引コスト実装方式

**CFD**:
```
position(t) = wn(t) × lev_mod(t) × L_s2(t)
daily_trade_cost(t) = |position(t) - position(t-1)| × spread_one_way
r_adjusted(t) = r_base(t) - daily_trade_cost(t)
NAV_adjusted = cumprod(1 + r_adjusted)
```

**DH ETF**:
```
daily_turnover(t) = |Δw_TQQQ| + |Δw_TMF| + |Δw_2036|  (ETF ドル取引額、レバ重み付け無し)
daily_trade_cost(t) = daily_turnover(t) × per_unit_cost
```
