## 1. 結論（要点）

1. **min(IS,OOS)⓽ 20%超・ハードベト無しの候補が確定**: 主推奨 **B3a_k365** = LU1強化マップ {Q0:1.4, Q1:1.4} × uniform 1.15 × C1現金利回り × >3x超過分くりっく株365 → **min⓽ +20.98%・WFA CI95_lo +22.52%・MaxDD −38.2%・6次元総合 8.28 > P09 8.12**。
2. 採用判定基準（総合 > 8.12 ∧ ベト無 ∧ min⓽ ≥ +18.5%）を満たすのは **B3a_k365 / B3c_k365 / LU2_C1_k365** の3候補。20%は**ベト緩和なしで**達成（B4の出番なし）。
3. **C1（bondOFF日の現金利回り）は全戦略に+0.25pp・ほぼノーリスク** → 採用変更と無関係に P09 にも適用推奨（実運用ガイド: OUT期 bondOFF 日の Bond枠は MMF/SBIハイブリッド預金等で SOFR 相当を確保）。
4. **C4（くりっく株365）で >3x 超過分コストが 2.5%→0.25%/yr に低下**。改善幅は LU2系 +0.26pp、B3系 +0.29〜0.43pp。spread感度（0.75→1.5pp）でも −0.14pp 以内で頑健。
5. **B4（条件付きブースト・post-hoc由来）は自然クローズ**: Stage2 上位8構成は全て B3 群で、B4 はフルゲートに残らなかった。post-hoc 仮説が事前登録仮説（B3）に勝てなかった、という正しい結末。
6. 採用変更（E4 → B3a_k365 等）は**ユーザー承認事項**。7月上旬の E4→P09 切替判断と同時に審議することを推奨（GAS は BOOST_MAP/STRATEGY CONFIG 切替で追従可・P09並走はそのまま）。

## 3. C1: bondOFF 日の現金利回り計上（実測）

- 対象: OUT期∧bondOFF の **3,061日（全営業日の23.2%）**。Bondスリーブ（重みw_b）のキャッシュ部分に日次SOFRを計上（スプレッド無し・MMF≈SOFR−5〜15bpなのでごく僅かに楽観だが1–2bp/yr水準）。
- **効果: min⓽ +0.25pp**（P09/LU1/LU2 とも +0.25〜0.26pp で一致）。計画想定の +0.5〜0.9pp より小さい。原因: min⓽ を規定する OOS期（2021-05〜）の前半は SOFR≈0% で、OOS加重平均金利が低いため。IS側には大きく効く（IS-OOS gap が約+1.0pp 拡大）— 実データ通りに正直に計上した結果であり、過大評価はない。
- 適用は全候補・全比較で統一（C1なし数値も CSV に保存済）。

## 4. B2-B4: レバ強化スイープ（44構成・2段階ゲート）

- **Stage1**（標準10指標＋ベト）: 44構成中 31 が min⓽≥19% 通過。**MaxDD−50% ベトに抵触する構成はゼロ**（最悪 −40.8%）— ベト交点はこのグリッド範囲の外。
- **Stage2**（フルゲート）: 上位8（全てB3群）が **8/8 PASS**。
- 構造観察（重要）: **q1（Q1ブースト 1.2→1.4）と uniform scale が主効果、q0 はほぼ非感応**（q0 1.4→2.0 で min⓽ +0.2pp 未満・WFE悪化 0.987→0.928）。応答曲面は滑らかで単調 — 「幸運なセル」ではない。
- **防御構造は不変**: 2000/2008/2020/2022 のストレス窓リターンは全構成で同一（+21.5%/+16.7%/+8.1%/+0.8%）。レバ強化はブル窓（IN期）でのみ発火し、OUT期の金/債防御層に触れない。悪化するのは 2015年型（worst CY −18.7%→−22.4%）。
- **B4（条件付きブースト LU3）**: post-hoc 由来仮説として厳格運用（posthoc_flag付与・単一OOS採用不可）。scale=1.0 では min⓽<19%、scale=1.15 でも +19.9% 止まりで **Stage2 の Top8 に残らずクローズ**。

## 8. 制約・正直な留意点（QC）

1. **bootstrap CI95_lo は全候補で負**（−2.8〜−3.3pp）: V7比の min⓽ 改善は95%水準で統計的有意でない。これは **P09 正典昇格時と同一の状況**（P09 も P=0.797・CI負で昇格）であり、⑥を6.6前後に抑える形で反映済。「有意な差」ではなく「一貫した優位の傾向＋フルゲート通過」が採用根拠である点は P09 と同じ。
2. **選択バイアス**: B3a は44構成の探索結果。緩和要因 = グリッドは計画書で事前登録（B3「本命」）・主効果(q1/scale)が単調で応答曲面が滑らか・q0非感応・フルゲート8/8PASS。それでも将来パフォーマンスは Stage2 表の点推定より CI95_lo 寄りに見るのが安全。
3. **2015年型の悪化**: worst暦年 −22.4%は P09 の −18.7% より深い。10年保有では W10★ +14.5% に回復するが、5年未満の取り崩し予定資金には不適。
4. **C1 の OOS 効果は小さい**（高金利期に効く設計。今後の金利低下局面では逓減）。
5. **くりっく株365の実務リスク**は §5 の5点。特に年次リセットの乗換オペレーションは年1回の手動作業。
6. E4 現行 Active（min⓽ +16.93% realistic）との比較では全合格候補が大幅優位だが、**E4→P09 切替判断（7月上旬予定）と一体で審議すべき**。GAS 並走（NASDAQ-strategy-gas）は本検証では未変更。
