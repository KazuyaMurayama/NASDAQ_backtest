## 解説（v3→v4 変更の意味）

### なぜ SOFR 補正が必要か

v3 では全戦略の SOFR 融資コストが未計上だった。TQQQ・TMF は証券貸借スワップを使った3倍レバレッジ商品であり、実際には毎日 **2×SOFR + 0.50% swap spread** の融資コストが発生する。v3 はこのコストを計上せず、CAGRを過大推計していた。

### 補正の根拠

- **TQQQ**: OLS 回帰 β_SOFR = -2.13 (R²=0.998) 実証済み
- **TMF**: 債券3倍レバ構造から 2×SOFR が理論的に導出
- **Gold 2x**: 先物ベース商品は1倍証拠金相当の SOFR が必要（2×ではなく1×）
- **債券モデル**: dgs10+dur7 は MAE 19.71%/yr、dgs30+dur17（時変）は 3.87%/yr — 後者が実測 TMF に最も近い

### 補正後の戦略序列

```

  2. DH Dyn 2x3x [A] : +22.50%（最高 Sharpe クラス 0.993）★ 推奨
  3. DH Dyn CAGR25+  : +21.39%（最良 Sharpe 1.062、MaxDD -29.6% 最小）
  4. DH Dyn 2x3x [B] : +19.44%
  5. Berkshire Hathaway: +19.47%（外部ベンチマーク）
  6. Ens2(Asym+Slope) : +17.50%

  9. BH 3x            :  +8.69%（無管理3倍、SOFR常時負担で最低）
```

> **BH 3x の大幅低下 (-10.52pp)** は示唆的: 「ただ3倍レバを持ち続けるだけ」では SOFR コストで侵食され、BH 1x にも大幅に劣後する。動的な CASH 退避が SOFR 補正後の重要性を持つことが明確になった。
