# Round 4 バックテスト結果サマリー

> **検証期間:** 1974年〜2021年（約47年間）
> **検証戦略数:** 55戦略（取引回数100回以内でフィルタリング後42戦略）
> **ベースレバレッジ:** 3倍（日次リバランス）

## 1. キーインサイト

### 発見1: Vol Spike検出が最も効果的な改善手法
```
DD(-18/92)+VT(25%)+VolSpike(1.5x)
- Sharpe: 0.896 (+4.7% vs ベースライン)
- CAGR: 31.45% (+5.7%)
- MaxDD: -58.51% (改善 +5.9%)
- Worst5Y: -8.02%
- 取引回数: 30回
```
**メカニズム:** ボラティリティが前日比1.5倍以上急騰した日はレバレッジを半減。暴落初期の大損失を回避。

### 発見2: 季節性フィルター（Q3/Q4縮小）がMaxDDを最も改善
```
DD(-18/92)+VT(25%)+Q3Q4weak(0.8x)
- Sharpe: 0.894 (+4.4%)
- MaxDD: -54.73% (最良、改善 +12.0%)
- Worst5Y: -7.23% (最良)
- 勝率: 70.21%
```
**メカニズム:** 7-10月（歴史的に弱い期間）のレバレッジを80%に縮小。

### 発見3: 非対称レバレッジが安定した改善を提供
```
DD(-18/92)+AsymLev(up30,dn20)
- Sharpe: 0.863 (+0.8%)
- CAGR: 30.24%
- MaxDD: -59.02% (+5.1%)
- 勝率: 70.21%
```
**メカニズム:** 上昇トレンド時Target Vol 30%、下落トレンド時20%で攻守切替。

### 発見4: 段階的リエントリーが取引回数を最小化
```
DD(-18/Staged)+VT(25%) 85-93
- 取引回数: 15回（最少、ベースライン比-50%）
- Sharpe: 0.860 (+0.5%)
- CAGR: 30.16%
```
**メカニズム:** 85%回復で50%ポジション、93%回復で100%に段階復帰。

### 発見5: アンサンブル戦略は過学習リスクを低減するが大幅改善はない
```
DynEns(60d)+VT(25%)
- Sharpe: 0.868 (+1.4%)
- 特徴: 複数DD戦略を過去60日パフォーマンスで動的加重
```

### 発見6: トリプルDDはデュアルDDに劣る
```
TripleDD(50/100/200) vs DualDD(100/200)
- TripleDD: Sharpe 0.807, 取引70回
- DualDD: Sharpe 0.825, 取引60回
→ 短期DD(50日)追加は過剰反応を招き逆効果
```

## 4. 推奨戦略ランキング

### 総合バランス型（推奨1位）
```
DD(-18/92)+VT(25%)+VolSpike(1.5x)
```
- 全指標でベースラインを上回る
- 実装シンプル（Vol前日比監視のみ追加）
- 取引回数維持（30回）

### 防御重視型（推奨2位）
```
DD(-18/92)+VT(25%)+Q3Q4weak(0.8x)
```
- MaxDD最良（-54.73%）
- Worst5Y最良級（-7.23%）
- 季節性ルールのみで実装容易

### 低回転型（推奨3位）
```
DD(-18/Staged)+VT(25%) 85-93
```
- 取引回数最少（15回、年0.3回）
- 段階的復帰で心理的負担軽減

### 分散型（推奨4位）
```
DynEns(60d)+VT(25%)
```
- 複数戦略の動的加重
- 単一パラメータ依存リスク低減
