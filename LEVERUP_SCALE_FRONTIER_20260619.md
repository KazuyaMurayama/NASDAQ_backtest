**読み方**:
- **min(IS,OOS)⓽ は 1.35→1.60 で単調上昇**（+23.83→+24.34→+25.31→+26.21%）＝リターン逓減ピーク未到達。採用ゲート min⓽≥+18.5% は全通過。
- **ハードベト判定**（MaxDD<−50%／WFE>1.5／Worst10Y★<0／Regime_min<−10%）: **scale1.35/1.40/1.50 は全PASS**。**scale1.60 は MaxDD−51.95%でベト抵触**（参考値）。
- **頑強性列は全scale構成 ⚠条件付**: WFE(0.90-0.94・過学習兆候なし)・CI95_lo(+27〜+31%・正で頑強)・CPCV p10(+18〜19%・正)・Regime_min(−5.4〜−7.7%・−10%まで余裕)は良好だが、**IS-OOS gap が 3.55〜4.90pp と 3pp 超**のため「✅頑強」でなく ⚠（高レバほど IS優位が拡大＝軽度の過学習/レジーム signal）。
- **Sharpe_Full はレバ上げで低下**（1.08→1.06、全て◎だが★のB3a 1.10未満）＝**純レバ（risk-adjusted は改善せず）**。P09_C1は Sharpe_Full +1.13（★）＝タイミング系でリスク効率最高。
- **最悪単日テールは比例超で悪化**（−18.7%→−22.4%、全て 2016 Brexit 日）。P09_C1の最悪単日は −13.82%（2000-01-04）と低レバを反映して軽微。
- **Worst5Y は scale1.50 でフラット割れ**（−0.02%）。P09_C1 は −0.58%（Worst5Y）と、低レバ構成にしては注意水準。
- ⚠ **対V7/対B3a bootstrap は全水準で95%非有意**（CI95_min 負: 対V7 −3.9〜−5.7pp）＝点推定は上だが「有意に上回る」とは言えない（B3a/P09昇格時と同じ家系的状況）。P09_C1はP_min=0.820で独立評価済み。
- **P09_C1 の実効L>3 日 ≈ 6.1%**（Q0:1.20/Q1:1.10ブーストで L=lev_raw×map×3 が3xを超える日。最大実効レバ約3.6x。正典 [LEVERUP_SWEEP §6.1](LEVERUP_SWEEP_RESULTS_20260612.md) の訂正値）だが、**P09のコストモデルは k365超過課金なし**（コスト計上上 excess_days=0）。「>3x日=0%」は旧（超過課金日数）定義で、実効L>3日数の6.1%が指標趣旨に忠実。

## 4. 脚注・前提
- **コスト**: TQQQ borrow=SOFR+0.50%・TER、>3x超過分 k365 EXCESS_EXTRA=0.25%/yr、売買コスト（Trades/yr連動）、C1 OUTフィル（Gold常時＋Bond@mom252>0・逆ボラ63日・TER込み）。全て日次でNAVに反映済。
- **§1 税後（⓽ ×0.8273）／§2・§3 税前**＝税基準が異なる（年次に税割付は不正確なため）。
- ⚠ **§1 頑強性セルの IS-OOS gap は税前CAGR差（pp）**＝他指標が税後⓽の中で gap のみ税前基準（gap自体に税変換は不要なため）。**税後差で見ると** scale系は約2.94〜4.05pp（独立QC再算）。判定（gap>3pp で⚠）は税前基準で行っている。
- **scale1.60 は MaxDD−51.95%で標準ハードベト抵触＝採用外の参考値**（[EVALUATION_STANDARD §3](EVALUATION_STANDARD.md)）。ユーザー選好で −50s% を許容し標準ベトを上書きする場合のみ検討対象。
- **NASDAQ 1倍B&H** はレバ無・コスト無・タイミング無のベンチマーク（指数 close）。
- **P09_C1 の >3x（実効L>3）日 ≈ 6.1%**（正典 [LEVERUP_SWEEP §6.1](LEVERUP_SWEEP_RESULTS_20260612.md) 訂正値）。ただし P09 のコストモデルは超過分に k365 課金をしていない（`leverup_b1c1_20260612.py` の excess_days=0＝コスト計上上はゼロ）。旧「>3x日=0%」は超過課金日数の定義で、**実効L>3日数の6.1%が指標趣旨に忠実**。
- **P09_C1 頑強性入力（§6.1/REGISTRY確定値）**: WFE=0.989・CI95_lo=+18.96%・CPCV=+14.15%・Regime_min=−0.08%・IS_OOS_gap=+2.56pp・対V7 P_min=0.820。
- 一次データ: `src/audit/leverext_high_stage1_20260618.py` / `audit_results/leverext_high_stage1_20260618.csv`（10指標・Stage-1）、`src/audit/leverup_b1c1_20260612.py` / `audit_results/leverup_b1c1_20260612.csv`（P09_C1 メトリクス・確定値）、`src/audit/annual_returns_scalefrontier_20260619.py` / `audit_results/annual_returns_scalefrontier_20260619.csv`（年次・scale系）。Stage-1 フルゲート詳細は [LEVERUP_EXTENSION_RESULTS_20260616.md](LEVERUP_EXTENSION_RESULTS_20260616.md)。P09_C1年次は同パイプライン（`_build_p09_on_base_c1`）で算出・本ファイルに直接埋め込み済み。
