# 計画: 2021年5月〜現在のOut-of-Sample検証

## ゴール
サンプルデータ（1974-2021/05）以降の実データで、MomDecel+Ens2(S+T)戦略の
月次リターンを算出し、B&H 1x / B&H 3x と比較する。

## ステップ

### Step 1: データ取得
- Yahoo Finance から NASDAQ Composite (^IXIC) の日次終値を取得
- 期間: 戦略のウォームアップに260営業日必要なため、2020年1月〜現在
  - ただし既存データ（1974-2021/05）と接続するため、既存データの末尾と重複する期間も取得して整合性を確認
- 方法: `yfinance` Pythonライブラリ（pip install yfinance）

### Step 2: データ結合
- 既存CSV（〜2021/05/07）と新規取得データ（2020/01〜現在）を接続
- 重複期間の終値が一致することを確認（データソース間のズレチェック）
- 結合後のCSVを保存（`NASDAQ_extended_to_2025.csv`）

### Step 3: 戦略計算
- 結合データに対して全4層を計算（DD, VT, SlopeMult, MomDecel）
- リバランス閾値20%を適用
- 5日遅延 + 年1.5%コストでバックテスト実行
- B&H 1x, B&H 3x も同条件で計算

### Step 4: 月次リターン算出
- Out-of-Sample期間: 2021年6月〜最新月
- 各月の月初NAV→月末NAVからリターンを算出
- 3戦略（MomDecel+Ens2, B&H 3x, B&H 1x）を並べて表示

### Step 5: 結果整理
- 月次リターン表（Out-of-Sample期間）
- 累積リターン推移
- サマリー統計（OOS期間のSharpe, CAGR, MaxDD）
- In-Sample vs Out-of-Sample の比較

## 注意事項
- ウォームアップ: 戦略の計算には260日のヒストリカルデータが必要
  → 2021/06のシグナルは2020/06頃からのデータで計算される
  → 既存データ（〜2021/05）がそのままウォームアップとして機能する
- Yahoo Financeのデータは調整済み終値（Adj Close）ではなく終値（Close）を使用
  → 既存データとの整合性のため
- NASDAQ Compositeはインデックスなので配当・分割の調整は不要
