# P09実戦略 5営業日後フォワードリターン ビン別レポート

作成日: 2026-06-12
最終更新日: 2026-06-12

> **目的**: GAS通知の「5営業日後フォワードリターン（過去統計）」を、旧Dyn2x3xレジーム表の流用から **P09_TQQQ実戦略のバックテストNAV（1974–2026・52.3年）に基づくビン別統計**へ置き換える。通知が示すその時々の状態（IN/OUT・mom63分位・レバ水準・Bondゲート）ごとに「この状態から5営業日先のP09リターンは過去どうだったか」を参照する。
> 一次根拠: `src/audit/p09_forward_return_20260612.py` / `audit_results/p09_forward_return_20260612.csv`

---

## 1. 手法

- **NAV**: P09_TQQQ（税前グロス・TQQQコスト・OUT期Gold/Bond充填込み）と LU1（CFD超過recost済）の日次NAV、1974-01-02〜2026-03-26。
- **フォワードリターン**: 各営業日 t から `NAV[t+5]/NAV[t]−1`（重複窓・記述統計）。
- **ビン（通知が表示する判断時点シグナルで分類）**:
  - IN（W1=HOLD）: mom63分位 Q0〜Q3（**前営業日値・凍結四分位境界**）× レバ帯（lo: lev_raw≤0.6 / hi: >0.6）= 8ビン
  - OUT（W1=OUT）: bond_mom252ゲート ON / OFF = 2ビン
- **表示値**: CAGR% = 平均5日対数リターンの年率化、med5d% = 5日リターン中央値、>0% = 勝率。

## 3. 読み方（重要な含意）

1. **ブーストの実証**: 最強ブースト局面 IN_Q0_hi（弱モメンタム=Q0でmap1.20/1.40が点灯し、かつ高レバ）は **5日後年率+121%（P09）/+146%（LU1）・勝率69%** と全ビン中圧倒的トップ。mom63逆張りブーストの設計意図がフォワードで裏付けられた。
2. **同じQ0でも低レバなら弱い**（+1.0%）: ブーストが効くのは「A2シグナルが高レバを許す＝下げ止まり確認後」の組み合わせ。Q0単独でなく **Q0×高レバ** が買い場シグナル。
3. **LU1の上乗せはまさにブースト局面で発生**: Q0_hi で P09+121%→LU1+146%（+25pp）。**「ブースト時だけCFDで上乗せする」LU1運用の合理性**を直接支持する。
4. **OUT期はbondゲートが効く**: bondON +10.6% vs bondOFF +1.3%。OFF時にBond枠をキャッシュに退避する設計は正当。
5. **注意**: 重複5日窓の記述統計であり独立サンプルではない（nは日数）。ビン間比較・桁感の把握用途に限定し、有意性主張には使わない。
