## 0. エグゼクティブサマリ（質問①②③への結論）

### Q1. いずれも同じ基準でバックテストされていたか → **NO（設計上3環境に分かれ、かつ表内で基準が混在）**
- 7戦略は **CFD（E4, vz065_l5, vz065_l7）/ ETF（DH-W1, V0, V7）/ 投信（P7）** の**3つの別コスト環境**。同一表に並ぶが、行間の直接比較は本来リンゴと別物。
- さらに**同一行内で CAGR=⓽税後 と Sharpe/MaxDD=ⓒ税前 が混在**（標準10指標の規定どおりだが、比較時に誤読を生む）。
- WFA窓数も出典により 49〜52窓と差。統一ハーネスで再計算すると全戦略 **52窓**で揃う。

### Q2. 商品コストを正しく反映できていたか → **NO（特にCFD財務コストが致命的に過少計上）**
- **CFD財務コストの過少計上（R7）が最大の問題**。旧 sim は `CFD_SPREAD_LOW=0.20%/yr`（くりっく株365）で計上していたが、SBI CFD の実勢は **(SOFR+3.0%)×レバL**。
- リポジトリは E4 のみ定額3.0%へ先行修正済（+27.41%→+20.0%）。だが**レバL比例×時変SOFR（Option A）で再計算すると更に低く、E4 CAGR_OOS は pretax +15.35%**。
- ETF/投信は実TER差（Gold 0.49→0.95%, TMF 0.91→1.06%）と売買コストのみで**現実コストでもほぼ不変**。

### Q3. 各数字は正しいか → **「正しい数字」と「誤った数字」が混在**
- **NAV・コスト後指標（MaxDD, Sharpe, Worst10Y, Trades実数）は正確に再現**（全戦略でハーネス再計算が表値と一致）。
- **ETF/投信のCAGR値は「税後値として正しい」が、税前他指標と混在表記**（R2）。pretax×0.8273で表値を再現可能（例: DH-W1 pretax+18.96%×0.8273=+15.69%≈表+15.74%）。
- **CFDのCAGR値は誤り**（過少計上）。リポジトリ修正後の E4 +20.0% でも、より厳密なrealisticでは pretax+15.35%。
- **DH-W1 の Trades/yr「68.7」は誤り**（NAV符号反転回数という疑似指標。実リバランスは17.6回）— R4。
- **副候補 l7 の CI95/WFE「N/A」は埋まった**（R9）。

### 最重要の含意：現実コスト下でランキングが逆転
旧表は CFD系（E4等）を比較基準=実質トップ扱いだったが、**現実コスト(Option A, pretax)では ETF/投信系が CFD系を上回る**:

## 4. 結論と推奨

1. **NAV計算ロジック自体は健全**（MaxDD/Sharpe/Worst10Y/実Tradesが全戦略で一致再現）。誤りは**コスト前提（CFD財務）**と**表記基準の混在**に集中。
2. **CFD戦略の魅力は現実コストで大きく失われる**（E4 OOS pretax +33.53%→+15.35%）。リポジトリの定額3.0%修正(+20.0%)は方向は正しいが**過小修正**で、レバL比例×時変SOFRでは更に低い。
3. **現実コスト下の実質ベストは ETF/投信環境**（P7・V7・DH-W1）。NISA非課税で更に優位。CFDはNISA不可・高財務コストで劣後。
4. **表の即時修正候補（報告のみ・正典未変更）**:
   - DH-W1 Trades/yr「68.7」→「17.6」（68.7はNAV符号反転の誤指標）
   - CAGR列に「⓽税後」を明示し、pretax ⓒ と分離（R2/R10）
   - vz065 ⚠行・E4行を realistic(Option A)基準で再掲
