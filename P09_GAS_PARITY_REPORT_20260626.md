## B. Bond配分の振動は「P09仕様どおり・バグではない」

### B.1 振動の事実
06-18以降、bond1x が **0.74 → 0.74 → 0 → 0 → 0.74 → 0** と数日おきに反転。同時に rebalance=YES が頻発。

### B.2 原因（生データから）
P09 の Bond ゲートは **`bond_on = (bond_mom252 > 0)`**（252営業日モメンタムの符号・**デッドバンドなし**）。直近のTLT（米国債20年超）が**1年前とほぼ同値**のため、bond_mom252 がゼロ近傍に張り付き、わずかな値動きで符号が反転している：

| 日付 | bond_mom252 | bond_on |
|---|---:|---|
| 06-17 | −0.0016 | 0 |
| 06-18 | **+0.0101** | 1 |
| 06-22 | +0.0029 | 1 |
| 06-23 | **−0.0065** | 0 |
| 06-25 | **+0.0070** | 1 |
| 06-26 | **−0.0006** | 0 |

10日間の bond_mom252 のレンジは **[−1.7%, +1.0%]**＝全て±2%以内。**TLTが約1年フラットという市場環境**が振動の真因で、計算は正しい。

### B.3 これは P09 の正典定義どおり
研究側 `run_p02_p09_backtest_20260611.py: _build_p09_nav` は `bond_on = np.where(np.isnan(bond_m252), False, bond_m252 > 0)`＝**ヒステリシスもデッドバンドも持たない単純符号ゲート**。GASはこれを忠実に再現している。**「P09の戦略に則っているなら問題ない」というユーザーの基準では問題なし。**

### B.4 ただし実務上の留保（運用摩擦）
- 振動は **OUT防御スリーブ内の Bond⇄現金の入れ替え**であり、株式（TQQQ）売買ではない。Bond⇄現金の往復は **取引コスト・スプレッドが相対的に小さい**（暴落時の株式売買ほど痛くない）。
- とはいえ「数日おきにTLT全売り→全買い」は実務では摩擦になる。これは**バックテストにも同じ振動が含まれており、その上で P09 はWFA/CPCV/bootstrap をPASSしている**＝振動込みで頑健性が確認済み。バックテストのTrades/yr（P09=29.2）にこの往復は計上済み。
- もし将来この往復を減らしたいなら **bond_mom252 に小デッドバンド（例 ±0.5%）またはヒステリシス**を入れる改良が考えられるが、それは**P09の定義変更＝別戦略**になるため、再WFA/CPCV検証が必要（現時点では未検証＝現運用は正典P09のまま正しい）。

## C. GAS採用戦略の妥当性（ユーザー3問への回答）

### C.1 GASが採用しているP09ロジック（=何を計算しているか）
**「素の P09（=P09_TQQQ）」**:
- DH-W1（Asymm Hysteresis, Enter≥0.7/Exit≤0.3）
- mom63 **デフォルトV7ブーストマップ {Q0:1.20, Q1:1.10, Q2:1.00, Q3:1.00}**・uniform scale **1.0**
- P09 OUT充填（Gold常時＋Bond@`bond_mom252>0`・逆ボラW63）
- **C1なし**（OUT∧bondOFF日の現金利回りSOFR計上は未実装）
- コスト非依存（通知専用なので≤3×/>3×の課金モデルは持たない）

### C.2 現行ベスト地形（CURRENT_BEST v8・2026-06-15確定）での相当
| 戦略 | 構成 | min⓽ | 位置づけ |
|---|---|---:|---|
| **B3a_k365**（**現ベスト**） | 強mapQ0/Q1=1.40 ×uniform1.15 +C1 +くりっく株365 | +20.98%(割引≈20.1%) | CAGR重視 |
| **P09_C1**（**ベスト候補**） | 素P09 **＋C1** | +17.77% | バランス・Sharpe最良級・CFD依存最小 |
| **GAS稼働中の「素P09」** | 素P09（C1なし） | +17.51% | **P09_C1 から C1 を抜いたもの** |

→ **GASのP09は、ベスト候補 P09_C1 の「土台」に相当**するが、**C1（現金利回り）が載っていない**ぶん min が約0.26pp低い（+17.51% vs +17.77%）。現ベストの B3a_k365 ではない。

### C.3 妥当性の判断材料（率直に）
**結論: 「Active候補を実運用前に検証する」目的としては妥当。ただし最新のベスト確定（v8）が未反映で、2つのギャップがある。**

1. ✅ **妥当な点**: GASのP09は、2026-06-11時点でActive候補だった戦略そのもの。並走の目的は「実運用前にシグナルが正しく出るか」の検証で、その用途には完全に機能している（10日間乖離ゼロ）。シグナルの骨格（DH-W1・mom63・OUT充填）は全ベスト候補で共通なので、検証価値は B3a/P09_C1 にも波及する。
2. ⚠ **ギャップ① C1未実装**: ベスト候補は P09**_C1**。C1（OUT∧bondOFF日にSOFR現金利回りを得る）は実装すれば min +0.26pp かつ**ほぼノーリスク**。今まさに振動でbondOFFになっている日（06-15/16/17/23/24/26）が現金待機なので、**C1の恩恵が直接効く局面**。→ GASに「bondOFF日はMMF/短期債で待機」と通知文へ明記する価値が高い。
3. ⚠ **ギャップ② 現ベストは B3a_k365**: より高いCAGR（min+20.98%）を狙うなら、ブーストマップを {1.40,1.40,1.05,1.00}・uniform×1.15 に変えるだけ（GASは `BOOST_MAP`＋scale の CONFIG 変更で追従可能）。ただし B3a は MaxDD−38%・worst暦年−22.4% と P09 より深く、**CAGR優先・DD許容のユーザー選好なら候補**だが「ベスト候補」としては Sharpe・CFD依存・2008防御で優れる **P09_C1 がユーザー確定の本線**。

### C.4 推奨
- **短期（すぐ）**: 現状の素P09で並走継続は問題なし（検証目的を満たす）。通知に「bondOFF日は現金をMMF/短期国債で待機（C1）」の一文を追加し、P09_C1相当の実運用にする。
- **中期（切替時）**: 実運用本採用の際に、ユーザーの最終選好で **P09_C1（バランス本線）** か **B3a_k365（CAGR攻め）** を選ぶ。どちらもGASは BOOST_MAP/scale/C1フラグの CONFIG 変更だけで追従できる（シグナル骨格は共通）。
