## 解釈とインサイト

### 1. =0%（CASH）vs =100%（フル投資）の対比が鮮明

| 指標 | =0% (CASH) | =100% (フル投資) |
|---|---|---|
| 構成比 | 20.3% | 25.1% |
| 5日プラス確率 | 51.7% | **62.0%** |
| 10日プラス確率 | 51.8% | **64.3%** |
| 20日プラス確率 | 51.7% | **67.0%** |
| 60日プラス確率 | 51.2% | **67.5%** |
| 250日プラス確率 | 66.1% | **78.7%** |
| 5日標準偏差 | 12.5% | **6.1%** |
| 10日標準偏差 | 17.4% | **9.0%** |
| 250日中央値 | +49.1% | **+41.9%** |

- =0%は**5日〜60日の全期間でプラス確率≒50%**（コイン投げ）かつ標準偏差が大きい → **退避判断は正しい**
- =100%は**5日でも62.0%、250日で78.7%** → 全期間で安定的にプラス → **投資判断の精度が高い**
- =0%の250日平均リターンが高いのは「暴落後の反発」を捉えているため（サバイバーシップ効果）

### 2. 0-10%ビン（N=66）：DD=HOLDだがレバレッジ極低

- サンプル数66と少数だが、短期5〜60日は全ビン中最悪のリターン
- **5日で-1.8%、10日で-7.5%** → DD退避はしないがVT/Slope/MomDecelが極端に低い → **暴落の入り口で最も危険な局面**
- 250日では+55.4%（プラス確率84.8%）に回復 → 嵐を耐えれば報われる

### 3. 短期リターンの有用性

- **5営業日**: =100%でも平均+1.4% → 年率換算で約+70%相当。ただし標準偏差6.1%
- **10営業日**: =100%で平均+2.8% → プラス確率64.3%
- **20営業日**: =100%で平均+5.2% → 安定的にプラス（プラス確率67%）
- **60営業日**: =100%で中央値+12.2% → 3ヶ月で二桁リターンが標準的

## 技術的補足

- **ビニング方式**: `raw_leverage == 0.0` → `=0%`, `raw_leverage >= 1.0` → `=100%`, それ以外は `int(val*10)` で10%刻み
- **前方リターン**: `Π(1 + 3×daily_return - 1.5%/252)` を各前方期間で累積
- **有効データ**: 前方5日は11,538日、前方250日は11,293日（末尾の前方データ不足分が減少）
- **ウォームアップ**: 400日（DD 200日 + SlopeMult/MomDecel 各200日の最大ウォームアップ期間）
- **GAS完全一致**: 全パラメータは [NASDAQ-strategy-gas](https://github.com/KazuyaMurayama/NASDAQ-strategy-gas) の `Code.gs` CONFIG と同一
