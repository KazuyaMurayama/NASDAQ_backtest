## 3. CFD コストモデルの根拠

### ボラティリティドラッグとは

日次リバランス型ETF (TQQQ) は毎日3倍固定を維持するため、価格変動ごとに
高値で買い・安値で売りをして vol drag が発生する:

```
D ≈ 0.5 × L × (L-1) × σ²
TQQQ (L=3, σ=22%): D ≈ 0.5 × 3 × 2 × 0.22² ≈ -14.52%/年
```

CFD/くりっく株365は毎日リバランスしないため、vol drag が発生しない。

### 線形コスト式

```python
# CFD の日次収益 (L = 3, 4, 5)
r_cfd = L * r_nasdaq - (L-1) * (sofr_daily + spread/252) - ter/252
```

レバレッジが高いほど借入コストが増加 (L-1 倍):

| レバレッジ | SOFR倍率 | SOFR=4%時の年率借入コスト (LOWスプレッド0.20%想定) |
|-----------|---------|--------------------------------------------------|
| 3x (TQQQ) | 2.0× | 8.0%+0.50%(swap)+0.86%(TER) = 9.36% |
| 3x (CFD)  | 2.0× | 8.0%+(2×0.20%) = 8.40% |
| 4x (CFD)  | 3.0× | 12.0%+(3×0.20%) = 12.60% |
| 5x (CFD)  | 4.0× | 16.0%+(4×0.20%) = 16.80% |

## 7. リスク所見

### 4x/5x のMaxDD・Worst5Y

- **DH Dyn 4x2x3x**: MaxDD = -49.98%, Worst5Y = +0.16%
- **DH Dyn 5x2x3x**: MaxDD = -54.87%, Worst5Y = -1.38%

### 実務上の制約

- **証拠金維持率**: くりっく株365は証拠金不足で追証 (5倍で暴落時ロスカット危険)
- **スリッページ**: CFDのスプレッドは市場状況で変動 (本シミュレーションは固定)
- **1980年代の高金利環境**: SOFR相当 15-20%時は借入コストが (L-1)×15-20% に拡大
- **ロールコスト**: 先物ベース商品は限月ロール時にコスト発生 (本シミュレーション未計上)
- **本シミュレーションはwn/wg/wbの再最適化をしていない**: 4x/5xでは現行重みが過大の可能性
