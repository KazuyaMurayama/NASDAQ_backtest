## 2. 判定の詳細

### 2.1 C1 — 名目固定の天井（条件付き・上限側を訂正）

**計算は正しい**: 白紙独立再実装（T1・既存コード不読）が全アンカーを4桁再現（0.8745 / 0.8965 / 0.9170 / 0.9995・cut2.4405 / 0.9945 / 0.9570）。**ブートストラップ方式にも頑健**（T2: block∈{3,4,5,6,8,10}・定常bs・部分史実・wrap禁止・horizon{15,25,30} の13スキーム全部で C1 の順位保存・全ペア p<0.0001）。

**しかし規約依存**（T3・5シード平均[min-max]）:

| 構成 | (a) lookahead | (b) lagged | (c) yearend |
|---|--:|--:|--:|
| 20:20 thr20（hold） | 0.8847 [0.8700-0.8965] | **0.7833** [0.7655-0.8000] | 0.7833 |
| 20:20 thr26（hold） | 0.9043 | 0.8343 | 0.7895 |
| 10:30 thr14（hold） | 0.9247 [0.9160-0.9315] | 0.8343 | 0.7278 |
| **20:20 thr20 hold なし** | **0.8785**（規約に依存しない） | 0.8785 | 0.8785 |

- lagged 下では「早め投入」系（thr>run0・補填厚め）は**すべて年0に全額投入＝「初日から全額運用」に退化して同値**（0.8343）。§3.2b の QC「動的早期投入(0.8965) > 静的全額運用(0.8285)」の +6.8pp は、**ルックアヘッドが作った差**（lagged では 0.8343 ≈ 静的 0.8285+乱数誤差）。
- **実装可能な名目固定の最良 = hold なし・thr=run0 の即投入（P=0.8785）**。奇しくも旧レポート（§3.2b 以前）の「天井≈0.88」に一致し、**0.917 への上方改訂の方がアーティファクト**だった。
- ルックアヘッドの水増し幅: 同一構成のペア比較で最大 **+19.7pp**（10:30 thr14、(a) vs (c): 不一致 1971 vs 2 経路）。

**結論**: 「名目固定では100%不能」は支持（強化）。「天井 0.917」は「実装不能な楽観上限」に格下げ。実装可能天井 ≈ **0.88**。

### 2.2 C2 — タイミング系全滅（名目=支持 / フロア=条件付きで覆る）

**名目固定世界**（T4・lagged 規約・実装可能27ルール: noHOLD thr/配分掃引・回復確認 L1/L2・実現DDトリガー20/30/40%・部分+早期2種・債券側条件・定期25%×4年）: **B0（noHOLD thr20 = 0.8785）を p<0.05 で上回るルールなし**（全て負または非有意）。C2 は公平な土俵でも成立。

**ただし §3.2b の中核脚注は覆る**: 「hold_if_crash の"非マイナスまで待つ"が正しい」は、lagged では **hold あり 0.7833 vs hold なし 0.8785（dP=−9.5pp、不一致 1 vs 953 経路、p=1.3e-284）**で真逆。「待つのが正しい」のは未来の符号を知っている場合だけで、実際に観測してから待つと回復年を踏み外す。

**フロア世界**（floor=3.6M・top_wr=0.16）では2ルールが基準（初日全額 0.9944）を有意超過:

| ルール | P（5シード） | ペア検定 | 実質終端中央値 |
|---|--:|--:|--:|
| 基準: 初日全額投入 | 0.9944 [0.9930-0.9960] | — | 220.7億 |
| F1: noHOLD thr20 | 0.9959 [0.9945-0.9970] | p=0.024 | 111.1億 |
| **F3: 実現DD30%で即投入** | **0.9983** [0.9980-0.9985] | **p=8.1e-8** | **67.4億** |

F3 は「運用スリーブがピーク比30%下落したら（回復を待たず）全弾投入」＝実装可能。P は改善するが**実質終端中央値を約 1/3 に削る**（暴落直後の高レバ投入は生存率を上げ資産成長を犠牲にする）。**「タイミングは全滅」はフロア世界では言い過ぎ**——ただし P↔終端のトレードオフ・ダイヤルであって無料の改善ではない。

### 2.3 C3 — フロア有無で最良投入ルールが逆転（覆る）

T5 細密グリッド（thr {12..30}×配分5段×**20シード**×2規約・floor3.6M・top0.16、pooled n=40,000）:

- **(a) lookahead**: 20:20 の thr≤20 が P≈0.9986–0.9990 の**プラトー**（thr18 と thr20 は 40,000 経路で不一致 0 件＝完全同一）、thr≥22 は初日全額に退化して 0.9951 に段差。Wilson CI 完全分離（0.9990 [0.9986-0.9993] vs 0.9951 [0.9944-0.9958]）。→ ルックアヘッド内では「予備温存が上」は維持（ただし「thr20 が唯一のピーク」ではない）。
- **(b) lagged**: **向きが反転**。初日全額 0.9942 [0.9934-0.9949] > 予備温存 20:20 thr20 0.9921 [0.9912-0.9929]（ペア検定 68 vs 152 経路、p=1.5e-8）。しかも lagged では初日全額が**終端も上**（2.15兆 vs 0.79兆）＝トレードオフでなく支配。
- **(c) yearend**: 同着（p=0.53）。

**判定**: 「フロアの有無で最良投入ルールが逆転する」という前セッションの中心発見は、**規約 (a) を採用した場合にのみ現れる**。V6-C 側の「補填厚め+早め投入が最良」は実装可能規約で消滅（§2.1）、V6-A 側の「現状 20:20・thr20 が最良」は lagged で反転・yearend で同着。**覆り**。ただし実務的な含意は限定的——V6-A 世界の構成間差は最大 0.2pp 程度で、**「フロア化そのもの」の支配的効果（+11pp）に比べれば微差**。

**条件付きの逆側の知見**（T6 補遺・最悪3年クラスタ 2014–2016（累積−38.0%）を引退直後に強制した条件付きストレス）: FA1（予備温存）0.8210 > FA2（初日全額）0.7829（lagged）。**無条件平均では初日全額が僅差で勝ち、暴落先頭条件では予備温存が勝つ**——どちらを重視するかの選択であり「最良」の一意な答えはない。

### 2.4 C4 — V6-A 改善不能・V6-A′（条件付き）

- **thr/配分では改善不能**: 支持。20シード細密グリッドで 0.9990 プラトーを超えるセルなし（lookahead）。lagged でも 0.9942–0.9943 が上限。
- **「保証フロア360万で P≥0.999」は規約依存**（T6 フロア劣化曲線・run20/res20 thr20 top0.16・全5シード P≥0.999 基準）:

| シナリオ | lookahead | lagged |
|---|--:|--:|
| ストレスなし | 3.2M（360万は 0.9991 でギリギリ） | **2.4M** |
| run −2pp | 2.8M | 2.0M |
| run −5pp | 2.4M | 2.0M |
| run−2pp+債券−1pp | 2.8M | 2.0M |

  実装可能規約では**無ストレスでも「P≥0.999 で保証できるフロア」は 240〜300万に低下**（360万のままなら P=0.9922 [0.9915-0.9930]＝「1000回の引退に8回弱」の労働）。
- **top_wr は効く（未掃引の穴・的中）**: フロア360万を維持したまま top_wr 0.16→0.12 で **lagged P 0.9921→0.9957・lookahead 0.9990→0.9994**（代償: 節約年 2.4→3.4年、終端微減）。さらに **0.16→0.18 に崖**（lagged 0.9921→0.9798）があり、現行 0.16 は崖の直前に立っている。**「改善不能」は thr/配分に限れば真、支出側ダイヤル（top_wr・フロア水準）と DD トリガー（F3）では改善可能**。
- **V6-A′（thr26M）は特別な点ではない**: thr∈{22,24,26,28,30} は全て「初日全額投入」に縮退して完全同値。「P−0.5pp で終端倍増」は thr>run0 の**単一の不連続スイッチ**の性質で、top_wr 0.14–0.18 の範囲では滑らかに成立（頑健）。thr26 という数字に意味はない。

### 2.5 C5 — フロア化が唯一の道（支持・強化）

- T2: 13スキーム中12で「フロア@40M > 名目@60M」保存（唯一の例外は 1975–2000 平穏部分史実での天井タイ 1.0000=1.0000、反転ではない）。
- T6: −2pp 世界でも FA1@40M（0.9976）> 60M名目（0.9770）> 49M名目（0.9474）、ペア検定 p≈1e-62〜1e-228。
- T6補遺: 最悪3年クラスタ先頭条件で名目固定は**投入ルール・規約を問わず壊滅**（NC1 lookahead 0.2655・lagged 0.0000、noHOLD 0.0000＝即投入が暴落に全弾吸われる）。フロア版のみ生存（FA1 lookahead 0.9707 / lagged 0.8210）。
- 部分史実 2000–2025（dotcom+GFC+2022 を含む）のみからの再サンプルでは名目 0.7084 vs フロア 0.9921＝差はむしろ拡大。
- **判定: 支持**。「複数の独立角度から反証を試みたが崩れなかった」と言える唯一の主張。

## 3. 方法論の再点検（T1 の残り）

- **R-STAT-1/2/3 照合**: P(労働=0) は経路依存**極値**（MaxDD型）ではなく**経路終端サバイバル統計**。run スリーブの年次負リターン連続長は史実最大 **2年**（クラスタ長 [1,1,1,1,1,1,2,1,2,1]）で block=5 ≥ クラスタ長のため、R-STAT-1 の「block≪危機長」違反ケースに該当しない。block を 3→10 に動かしても P は滑らかに変化（不連続ジャンプなし）。**ブートストラップ設計は妥当**。
- **P(labor0)==P(ruin0) の恒等**（名目固定）: 白紙再実装の全 variant・全構成で成立、1経路トレースでも確認（払えない年→全額支出→資産0→以後 labor 連鎖）。フロア版では分離（同一経路で名目=labor11/ruin、フロア=labor0/cut10）。**定義上の帰結でありバグでない**——支持。
- **最悪10年窓は識別力なし**: sc2.6 の最悪10年（2007–2016）ですら**累積+663.9%**＝この高レバ系列に「負の10年」は史実上存在しない。10年窓ストレスは全構成 P=1.0 で無意味（設計の限界として記録）。効くのは3年クラスタ（§2.5）。
- **表示単位バグ（既存ハーネス・数値は正しい）**: `labor_zero_v6a_grid_20260701.py` と `labor_zero_v6_thr_dial_20260701.py` の一部プリントが「oku」表記だが実値は百万円（例: 11767oku→実際は 11,767M=117.7億）。正典レポート本文の億表記は正しい（換算済）。プリントラベルのみの問題。

## 4. 実運用への含意（採否はユーザー判断・分析であり本番助言でない）

1. **V6-A（フロア化）の採用判断は揺るがない**。フロア化の主効果（名目比 +11pp・テール保全・暴落先頭でも生存）は全攻撃に耐えた。
2. **ただし性能の読みは下方修正**: 実装可能規約での V6-A は **P≈0.992（360万フロア・top0.16）**。「99.85〜100%」はルックアヘッド値。**P≥0.999 が必要なら (i) top_wr を 0.12 に絞る（P 0.9957・節約年+1年）か (ii) フロアを 240〜300万に下げる**。
3. **投入ルールの推奨を簡素化できる**: 実装可能規約では V6-A 世界の投入ルール差は 0.2pp。無条件平均なら「初日から全額運用+フロア支出」が最良（管理も最簡）だが、**引退直後の暴落を重く見るなら予備温存（現行 20:20・thr20）**。「非マイナス年まで待つ」ルールに実装可能な優位性はない。
4. **V6-C（名目固定堅持）の実装可能な最良は「hold なし・運用が初期額を割ったら即・全額投入」で P≈0.88**。§0.1 の「補填厚め10:30」・§0.3 の「早め投入・待機」推奨は実装可能規約では根拠を失う。ただし即投入は暴落先頭条件で全滅する（フロア化しない限りテールは救えない——C5）。

## 5. 方法論教訓（今後の検証で必ず適用）

1. **年次粒度シミュレーションの意思決定ルールは「当期リターン参照」のルックアヘッドを必ず監査せよ**。ルールが `r[k]` を見てから年 k の行動を決めていないか。lagged / yearend 規約で感度を出し、実装可能側の値を主報告とする。（EVALUATION_STANDARD §4.4 に R-STAT-4 として追加）
2. **「効くダイヤル」を発見したときこそ情報規約を疑え**。前回教訓「効かないと言う前に掃引したか」の対偶——thr ダイヤルは掃引の結果「効いた」が、その効きは未来情報の漏れだった。
3. N=2000 で 0.5pp を語るときは**同一経路ペア検定**（McNemar）を使う。独立シード比較では分解能不足。
4. サバイバル系ストレスは「最悪10年」より「**最悪の短期クラスタを引退直後に強制**」が識別的（高レバ・モメンタム系列には負の10年が存在しないことがある）。
