# B2: k_lt スイープ — LT2 バイアス強度の最適化

**実行日**: 2026-05-21
**目的**: LT2 バイアス強度 k_lt を {0.3, 0.5(REF), 0.7, 1.0} でスイープし、k_lt=0.5 (B1 確定値) が最適かどうかを確認する

---

## 1. セットアップ

### スイープ設定

- **k_lt グリッド**: [0.3, 0.5, 0.7, 1.0]
- **LT2 N**: 750 日
- **S2 固定パラメータ**: target_vol=0.8, k_vz=0.3, gate_min=0.5, n=20, l_min=1.0, l_max=7.0
- **CFD スプレッド**: CFD_SPREAD_LOW
- **IS 期間**: 1974-01-02 〜 2021-05-07
- **OOS 期間**: 2021-05-08 〜

### lt_bias の式

```
lt_sig  = compute_lt2(close, N=750)
lt_bias = signal_to_bias(lt_sig, k_lt)  # = (-k_lt * 0.5 * z).clip(-0.5, +0.5)
lev_mod = clip(lev_A + lt_bias, 0, 1)
```

## 5. 考察

### k_lt と lt_bias の関係

| k_lt | lt_bias 式 | バイアス最大幅 |
|-----:|-----------|------------|
| 0.3 | (−0.15×z).clip(−0.5,+0.5) | ±0.15 |
| 0.5 | (−0.25×z).clip(−0.5,+0.5) | ±0.25 (REF) |
| 0.7 | (−0.35×z).clip(−0.5,+0.5) | ±0.35 |
| 1.0 | (−0.50×z).clip(−0.5,+0.5) | ±0.50 |

k_lt が大きいほど LT2 の逆張りバイアスが強まり、クランプ発生頻度が増加する。
MaxDD・Worst5Y への影響を確認すること。

---

## 6. 結論

**k_lt=0.5 が引き続き最良。** k_lt=0.5 を試みたが Sharpe 改善が基準 (+0.020) に達しないため B1 ベスト (k_lt=0.5) を維持する。
