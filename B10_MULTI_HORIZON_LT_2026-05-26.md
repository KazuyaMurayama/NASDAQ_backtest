## §1 目的・設計

Round 1A クリティカルシンキングでの発見: LT2 N-スイープが
**双峰分布** (N=600〜1250 が谷、N=750 と N=1500 にピーク) を示すこと
に対して、**複数窓の z-score を合成して双峰の両ピークを 1 本に統合**
したロバストなシグナルを構築・検証する。

### 設計

```
lt_sig_multi(t) = Σ_i w_i * z_i(t)         # z_i = compute_lt2(close, N_i)
                  clip to [-3.0, +3.0]

# E4 Regime k_lt 採用 config 固定
vz(t) > +0.7  → k(t) = 0.8
vz(t) < -0.7  → k(t) = 0.1
otherwise     → k(t) = 0.5
lt_bias(t)    = (-k(t) * lt_sig_multi(t) * 0.5).clip(-0.5, 0.5)
```

- **IS 期間**: 1974-01-02 〜 2021-05-07
- **OOS 期間**: 2021-05-08 〜
- **データ年数**: 52.26 年

## §4 結論・次ステップ

### 最良マルチホライズン候補 vs ベースライン (E4 + LT2-N750) との差分

- **最良マルチホライズン**: 2窓 N=750+1500 equal (verdict: **WARN**)
- ΔSharpe_OOS = **-0.019** (mh=+0.872 vs ref=+0.891)
- ΔCAGR_OOS = **-1.65pp** (mh=+31.88% vs ref=+33.53%)
- ΔMaxDD = **-1.40pp** (mh=-61.41% vs ref=-60.01%)
- IS-OOS gap = **+0.16pp**
- Trades/yr = **27.1** (ベースラインと同等の連続性)

### 総合判定

**マルチホライズン候補: PASS=0, WARN=1**。
複数窓 z-score 合成は現行ベストと同等水準（差 ≤ 0.02 Sharpe）に達するが、
明確な改善は得られない。Round 1A の N=1500 単独優位は LT2 単一窓段階での結果であり、
E4 Regime k_lt 適用後はベスト N=750 を上回らない（N=750 + E4 の相性が際立つ）。

WARN configs (マルチホライズン):
- 2窓 N=750+1500 equal → Sharpe_OOS=+0.872, CAGR_OOS=+31.88%, gap=+0.16pp

### Round 1A 仮説の評価

Round 1A は「N=750 と N=1500 の双峰分布 → 2つの異なるレジームに偶然フィット」
という過学習仮説を提示。本ラウンドの結果:

- **2窓 N=750+1500 equal**: Sharpe_OOS=+0.872 (ベースライン比 -0.019)
  - IS-OOS gap = +0.16pp（ベースライン -1.81pp より大）
  - 双峰合成によりOOSは改善せず、ベースラインの優位は偶然ではなく**N=750 単体の構造的優位**を示唆

結論: **Round 1A の「双峰=過学習」仮説は本ラウンドでは支持されない**。
N=750 + E4 の組み合わせは合成によって希釈されるほどの構造的シグナルを持つ。

### Round 2C (E5 チーム) へのインプリケーション

- 本ラウンドでは **E4 Regime k_lt** を採用 config (k_lo=0.1, k_hi=0.8, vz_thr=0.7) で固定
- マルチホライズン合成シグナルは E4 と独立した直交軸であり、E5 拡張時のレジーム設計
  でも合成シグナルを基底に使える可能性 (例: VZ レジーム以外のレジーム — 経済サイクル、
  金利環境など — に対するマルチホライズン LT2 の挙動評価)
- Sharpe 差分が小さい場合は E5 でレジーム軸を増やす方が有効と推定

### Round 2B (F10 チーム) へのインプリケーション

- F10 (ファクター拡張) 系は VZ/LT2 以外の方向 (例: VIX, クレジットスプレッド, etc.) を
  追加することで本ラウンドの限界を補完しうる
- 本ラウンドの結果は「LT2 内部の窓多様化だけでは Sharpe 0.02 改善が困難」を示すため、
  F10 で **異質シグナル** の追加価値を測る基準となる

### WFA 実施推奨

**推奨: WFA スキップ**（マルチホライズン候補が Sharpe 改善 +0.020 未達のため進格基準不達）。
現行ベスト (E4 + LT2-N750) を維持し、Round 2B (F10) / 2C (E5) の結果を待って判断。
