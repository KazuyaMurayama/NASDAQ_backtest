## 概要

DH Dyn 2x3x [A] に対して Top5 シグナルの組合せゲートを適用したバックテスト結果。

- IS:  1974-01-02 〜 2021-05-07
- OOS: 2021-05-08 〜 2026-03-26
- Crash years: [1981, 1988, 1994, 2015, 2022]
- 採用基準: Worst5Y 改善幅最大 (subject to CAGR_OOS ≥ 20%)

### ゲート合成設計
```
NAS-gate signals (HY/YC/CPI/MA): 乗算合成
  wn = clip(wn_A * ∏nas_gates, 0, 1)
Bond-gate signal (Dyn_Corr)     : wb のみ削減
  rest = 1 - wn
  wg   = rest * 0.5
  wb   = rest * 0.5 * bond_gate
```
