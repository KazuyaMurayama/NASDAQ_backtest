## 📋 §0 検証目的

DH-Z2 (v4 採用版、F10 ε + binary vz HOLD/OUT) は DH-REF (現行) に対し **OOS CAGR を +2.70pp 改善** したが、**IS CAGR が -5.60pp 大幅低下** (IS+20.02% → IS+14.42%) という trade-off があった。さらに Worst10Y も +12.57% → +6.36% に劣化。

→ DH のもつ「IS 強さ」(genuine alpha + 防御性能) を温存しつつ、OOS gap も改善する 3 候補を検証:

| ID | 機構 | 狙い |
|---|---|---|
| **DH-W1** | Asymmetric + Hysteresis (Enter `lev_mod_065 ≥ 0.7`, Exit `≤ 0.3`, 中間は現状態維持) × DH base 配分 | Z2 の whipsaw (2021/2025 で bull を逸失) を解消 |
| **DH-W2** | Z2 と同じ binary vz HOLD mask、ただし **OUT 時 100% TMF** (cash でなく) | risk-off 時の bond rally を取りに行く |
| **DH-W3** | 3-state preset switch: 高確信 80/10/10、低 100% TMF、中間 DH base | preset 切替で IS bull capture + OOS bear defense 両立 |
